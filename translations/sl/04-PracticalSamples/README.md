# Praktične uporabe in projekti

## Kaj boste naučili
V tem razdelku bomo prikazali tri praktične aplikacije, ki prikazujejo vzorce razvoja generativne umetne inteligence z Javo:
- Ustvarite večmodalni generator zgodb o hišnih ljubljenčkih, ki združuje AI na strani odjemalca in strežnika
- Implementirajte lokalno integracijo AI-modelov s predstavitvijo Foundry Local Spring Boot
- Razvijte storitev Model Context Protocol (MCP) z uporabo primera kalkulatorja

## Uvod

To poglavje prikazuje **vzorce razvoja generativne AI z Javo** s pomočjo vzorčnih projektov. Vsak projekt je popolnoma funkcionalen in prikazuje specifične AI tehnologije, arhitekturne vzorce in najboljše prakse, ki jih lahko prilagodite za svoje aplikacije.

### Foundry Local Spring Boot predstavitev

**[Foundry Local Spring Boot predstavitev](foundrylocal/README.md)** prikazuje, kako integrirati lokalne AI modele z uporabo **OpenAI Java SDK**. Predstavlja povezavo z modeli, ki tečejo na Foundry Local (npr. **Phi-4-mini**), z avtomatskim zaznavanjem modela, kar omogoča izvajanje AI aplikacij brez odvisnosti od oblačnih storitev.

### Generator zgodb o hišnih ljubljenčkih

**[Generator zgodb o hišnih ljubljenčkih](petstory/README.md)** je privlačna spletna aplikacija Spring Boot, ki prikazuje **večmodalno AI obdelavo** za ustvarjanje ustvarjalnih zgodb o hišnih ljubljenčkih. Združuje AI zmogljivosti na strani odjemalca in strežnika z uporabo transformer.js za AI interakcije v brskalniku in OpenAI SDK za obdelavo na strežniku.

### MCP storitev kalkulatorja (prijazna predstavitev MCP za začetnike)

**[MCP storitev kalkulatorja](calculator/README.md)** je preprost prikaz **Model Context Protocol (MCP)** s Spring AI. Nudi prijazno uvodno predstavitev konceptov MCP, prikazuje, kako ustvariti osnovni MCP strežnik, ki komunicira z MCP odjemalci.

## Napredek učenja

Ti projekti so zasnovani tako, da gradijo na konceptih iz prejšnjih poglavij:

1. **Začnite preprosto**: Začnite s predstavitvijo Foundry Local Spring Boot za razumevanje osnovne AI integracije z lokalnimi modeli
2. **Dodajte interaktivnost**: Nadaljujte z Generatorjem zgodb o hišnih ljubljenčkih za večmodalne AI in spletne interakcije
3. **Naučite se osnov MCP**: Preizkusite MCP storitev kalkulatorja za razumevanje osnov Model Context Protocola

## Povzetek

Čestitke! Zdaj ste raziskali nekaj pravih aplikacij:

- Večmodalne AI izkušnje, ki delujejo tako v brskalniku kot na strežniku
- Lokalna integracija AI modelov z uporabo sodobnih Java ogrodij in SDK-jev
- Vašo prvo storitev Model Context Protocol, ki prikazuje, kako se orodja povezujejo z AI
