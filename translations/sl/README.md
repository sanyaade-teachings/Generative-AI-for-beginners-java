## Struktura tečaja in učna pot

### **Poglavje 1: Uvod v generativno AI**
- **Osnovni koncepti**: Razumevanje velikih jezikovnih modelov, tokenov, vdelav in zmogljivosti AI
- **Java AI ekosistem**: Pregled Spring AI in OpenAI SDK-jev
- **Protokol konteksta modela**: Uvod v MCP in njegova vloga v komunikaciji AI agentov
- **Praktične uporabe**: Resnični scenariji, vključno s klepetalnicami in generiranjem vsebine
- **[→ Začnite s Poglavjem 1](./01-IntroToGenAI/README.md)**

### **Poglavje 2: Nastavitev razvojnega okolja**
- **Azure AI Foundry**: Zagotovite nameščanje modelov kot kodo z Bicep in Azure Developer CLI (azd)
- **Spring Boot + Spring AI**: Najboljše prakse za razvoj podjetniških AI aplikacij
- **Avtentikacija brez ključev**: Varna povezava z Microsoft Entra ID — brez upravljanja API ključev
- **Razvojna orodja**: Docker kontejnerji, VS Code in konfiguracija GitHub Codespaces
- **[→ Začnite s Poglavjem 2](./02-SetupDevEnvironment/README.md)**

### **Poglavje 3: Osnovne generativne AI tehnike**
- **Prompt inženiring**: Tehnike za optimalne odzive AI modelov
- **Vdelave in vektorske operacije**: Implementacija semantičnega iskanja in ujemanja podobnosti
- **Generacija s podporo iskanja (RAG)**: Združevanje AI z vašimi lastnimi podatkovnimi viri
- **Klic funkcij**: Razširite zmogljivosti AI s prilagojenimi orodji in vtičniki
- **[→ Začnite s Poglavjem 3](./03-CoreGenerativeAITechniques/README.md)**

### **Poglavje 4: Praktične uporabe in projekti**
- **Generator zgodb o hišnih ljubljenčkih** (`petstory/`): Kreativno generiranje vsebin z Azure AI Foundry
- **Foundry lokalna demonstracija** (`foundrylocal/`): Lokalna integracija AI modela z OpenAI Java SDK
- **MCP storitev kalkulatorja** (`calculator/`): Osnovna implementacija protokola Model Context z Spring AI
- **[→ Začnite s Poglavjem 4](./04-PracticalSamples/README.md)**

### **Poglavje 5: Odgovorni razvoj AI**
- **Azure AI Foundry varnost vsebine**: Preizkusite vgrajeno filtriranje vsebine in varnostne mehanizme (trdi bloki in mehki zavrnitve)
- **Demonstracija odgovorne AI**: Praktičen primer delovanja sodobnih varnostnih sistemov AI
- **Najboljše prakse**: Ključna navodila za etičen razvoj in uvajanje AI
- **[→ Začnite s Poglavjem 5](./05-ResponsibleGenAI/README.md)**
