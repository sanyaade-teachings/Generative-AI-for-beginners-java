# ಪ್ರಾರಂಭಿಕರಿಗಾಗಿ ಜನರೇಟಿವ್ ಏಐ - ಜಾವಾ ಆವೃತ್ತಿ
[![Microsoft Foundry Discord](https://dcbadge.limes.pink/api/server/nTYy5BXMWG)](https://discord.gg/nTYy5BXMWG)

![ಜನರೇಟಿವ್ ಏಐ ಫಾರ್-BEGINನರ್ಸ್ - ಜಾವಾ ಆವೃತ್ತಿ](../../translated_images/kn/beg-genai-series.8b48be9951cc574c.webp)

**ಸಮಯ ಬದ್ಧತೆ**: ಸಂಪೂರ್ಣ ವರ್ಕ್‌ಶಾಪ್ ಅನ್ನು ಆನ್‌ಲೈನ್‌ನಲ್ಲಿ ಸ್ಥಳೀಯ ಸೆಟಪ್ ಇಲ್ಲದೆ ಪೂರ್ಣಗೊಳಿಸಬಹುದು. ಪರಿಸರ ಸೆಟಪ್ 2 ನಿಮಿಷಗಳಲ್ಲಿದ್ದು, ಮಾದರಿಗಳನ್ನು ಅನ್ವೇಷಿಸುವುದು ಅನ್ವೇಷಣಾ ಗಂಭೀರತೆ ಹೊಂದಿಕೆಯಿಂದ 1-3 ಗಂಟೆಗಳಾಗುತ್ತದೆ.

> **ವೇಗದ ಪ್ರಾರಂಭ**

1. ಈ ರೆಪೊಸಿಟರಿಯನ್ನು ನಿಮ್ಮ GitHub ಖಾತೆಗೆ ಫೋರ್ಕ್ ಮಾಡಿ
2. **Code** → **Codespaces** ಟ್ಯಾಬ್ → **...** → **New with options...** ಕ್ಲಿಕ್ ಮಾಡಿ
3. ಡಿಫಾಲ್ಟ್‌ಗಳನ್ನು ಬಳಸಿರಿ – ಇದರಿಂದ ಈ ಕೋರ್ಸ್‌ಗಾಗಿ ರಚಿಸಲಾದ ಡೆವೆಲಪ್‌ಮೆಂಟ್ ಕಂಟೈನರ್ ಆಯ್ದು கொள்ளುತ್ತದೆ
4. **Create codespace** ಕ್ಲಿಕ್ ಮಾಡಿ
5. ಪರಿಸರ ಸಿದ್ಧವಾಗಲು ~2 ನಿಮಿಷ ಕಾಯಿರಿ
6. ನೇರವಾಗಿ [ಮೊದಲ ಉದಾಹರಣೆ](./02-SetupDevEnvironment/README.md#step-2-create-a-github-personal-access-token) ಗೆ ಹೋಗಿ

## ಕೋರ್ಸ್ ರಚನೆ ಮತ್ತು ಕಲಿಕೆಯ ಮಾರ್ಗ

### **ಅಧ್ಯಾಯ 1: ಜನರೇಟಿವ್ ಏಐ ಗೆ ಪರಿಚಯ**
- **ಮೂಲಭೂತ ಕಲ್ಪನೆಗಳು**: ವಿಶೇಷವಾಗಿ ಬೃಹತ್ ಭಾಷಾ ಮಾದರಿಗಳು, ಟೋಕನ್ಗಳು, embeddings ಮತ್ತು ಏಐ ಸಾಮರ್ಥ್ಯಗಳನ್ನು ಅರ್ಥಮಾಡಿಕೊಳ್ಳುವುದು
- **ಜಾವಾ ಏಐ ಇಕೋಸಿಸ್ಟಮ್**: ಸ್ಪ್ರಿಂಗ್ ಏಐ ಮತ್ತು ಓಪನ್‌ಎಐ SDKಗಳ ಅವಲೋಕನ
- **ಮಾಡಲ್ ಕಂಟೆಕ್ಸ್ಟ್ ಪ್ರೋಟೋಕಾಲ್**: MCPಗೆ ಪರಿಚಯ ಮತ್ತು ಏಐ ಏಜೆಂಟ್ ಸಂವಹನದಲ್ಲಿ ಇದರ ಪಾತ್ರ
- **ಪ್ರಾಯೋಗಿಕ ಅನ್ವಯಗಳು**: ಚಾಟ್‌ಬಾಟ್‌ಗಳು ಮತ್ತು ವಿಷಯ ಸೃಷ್ಟಿಕರ್ತ ಸಂಧರ್ಭದಲ್ಲಿ ನೈಜ ಜಗತ್ತಿನ ಉದಾಹರಣೆಗಳು
- **[→ ಅಧ್ಯಾಯ 1 ಪ್ರಾರಂಭಿಸಿ](./01-IntroToGenAI/README.md)**

### **ಅಧ್ಯಾಯ 2: ಡೆವೆಲಪ್‌ಮೆಂಟ್ ಪರಿಸರ ಸೆಟಪ್**
- **ಆಜ್ಯೂರ್ ಏಐ ಫೌಂಡ್ರಿ**: Bicep ಮತ್ತು Azure ಡೆವೆಲಪರ್ CLI (azd) ಬಳಸಿ ಮಾದರಿ ನಿಯೋಜನೆಗಳನ್ನು ಕೋಡ್ ರೂಪದಲ್ಲಿ ಒದಗಿಸಿ
- **ಸ್ಪ್ರಿಂಗ್ ಬೂಟ್ + ಸ್ಪ್ರಿಂಗ್ ಏಐ**: ಎಂಟರ್‌ಪ್ರೈಸ್ ಏಐ ಅಪ್ಲಿಕೇಶನ್ ಅಭಿವೃದ್ಧಿಗೆ ಉತ್ತಮ ಅಭ್ಯಾಸಗಳು
- **ಕೀಲ ರಹಿತ ಪ್ರಾಮಾಣೀಕರಣ**: ಮೈಸ್ರೋಸಾಫ್ಟ್ ಎಂಟ್ರಾ ID ಇತ್ತೀಚರಿಸುವ ಮೂಲಕ ಸುರಕ್ಷಿತ ಸಂಪರ್ಕ – API ಕೀಲಿಗಳು ನಿರ್ವಹಣೆ ಅಗತ್ಯವಿಲ್ಲ
- **ಡೆವೆಲಪ್‌ಮೆಂಟ್ ಸಲಕರಣೆಗಳು**: ಡಾಕರ್ ಕಂಟೈನರ್, VS ಕೋಡ್ ಮತ್ತು GitHub Codespaces ಕನ್ಫಿಗರೇಶನ್
- **[→ ಅಧ್ಯಾಯ 2 ಪ್ರಾರಂಭಿಸಿ](./02-SetupDevEnvironment/README.md)**

### **ಅಧ್ಯಾಯ 3: ಪ್ರಮುಖ ಜನರೇಟಿವ್ ಏಐ ತಂತ್ರಗಳು**
- **ಪ್ರಾಂಪ್ಟ್ ಇಂಜಿನಿಯರಿಂಗ್**: ಅತ್ಯುತ್ತಮ ಏಐ ಮಾದರಿ ಪ್ರತಿಕ್ರಿಯೆಗಳ ತಂತ್ರಗಳು
- **Embeddingಗಳು ಮತ್ತು ವೆಕ್ಟರ್ ಕಾರ್ಯಾಚರಣೆಗಳು**: ಅರ್ಥಪೂರ್ಣ ಹುಡುಕಾಟ ಮತ್ತು ಸಮಾನತೆ ಹೊಂದಾಣಿಕೆ ಅನುಷ್ಠಾನ
- **ರಿಟ್ರೀವಲ್-ಆಗ್ಮೆಂಟೆಡ್ ಜನರೇಶನ್ (RAG)**: ನಿಮ್ಮದೇ ಡೇಟಾ ಮೂಲಗಳೊಂದಿಗೆ ಏಐ ಸಂಯೋಜನೆ
- **ಫಂಕ್ಷನ್ ಕಾಲಿಂಗ್**: ಕಸ್ಟಮ್ ಸಾಧನಗಳು ಮತ್ತು ಪ್ಲಗಿನ್‌ಗಳ ಮೂಲಕ ಏಐ ಸಾಮರ್ಥ್ಯ ವಿಸ್ತರಣೆ
- **[→ ಅಧ್ಯಾಯ 3 ಪ್ರಾರಂಭಿಸಿ](./03-CoreGenerativeAITechniques/README.md)**

### **ಅಧ್ಯಾಯ 4: ಪ್ರಾಯೋಗಿಕ ಅನ್ವಯಗಳು ಮತ್ತು ಪ್ರಾಜೆಕ್ಟ್‌ಗಳು**
- **ಪೇಟು ಕಥಾ ಸೃಷ್ಠಿಕರ್ತ** (`petstory/`): ಆಜ್ಯೂರ್ ಏಐ ಫೌಂಡ್ರಿಯೊಂದಿಗೆ ರಚನಾತ್ಮಕ ವಿಷಯ ಸೃಷ್ಟಿ
- **ಫೌಂಡ್ರಿ ಸ್ಥಳೀಯ ಡೆಮೊ** (`foundrylocal/`): ಓಪನ್‌ಎಐ ಜಾವಾ SDK ಸಹಿತ ಸ್ಥಳೀಯ ಏಐ ಮಾದರಿ ಏಕರೂಪತೆ
- **MCP ಕ್ಯಾಲ್ಕುಲೇಟರ್ ಸೇವೆ** (`calculator/`): ಸ್ಪ್ರಿಂಗ್ ಏಐ ಬಳಸಿ ಮೂಲ Model Context Protocol ಅನುಷ್ಠಾನ
- **[→ ಅಧ್ಯಾಯ 4 ಪ್ರಾರಂಭಿಸಿ](./04-PracticalSamples/README.md)**

### **ಅಧ್ಯಾಯ 5: ಜವಾಬ್ದಾರಿಯುತ ಏಐ ಅಭಿವೃದ್ಧಿ**
- **ಆಜ್ಯೂರ್ ಏಐ ಫೌಂಡ್ರಿ ವಿಷಯ ಸುರಕ್ಷತೆ**: ಆಂತರಿಕ ವಿಷಯ ಸಂಯೋಜನೆ ಮತ್ತು ಸುರಕ್ಷತೆ ಯಂತ್ರಗಳನ್ನು ಪರೀಕ್ಷಿಸಿ (ಕಠೋರ ಅಡ್ಡಿಗಳು ಮತ್ತು ನರ್‌ಮಾಳಿ ನಿರಾಕರಣೆಗಳು)
- **ಜವಾಬ್ದಾರಿಯುತ ಏಐ ಡೆಮೊ**: ಆಧುನಿಕ ಏಐ ಸುರಕ್ಷತಾ ವ್ಯವಸ್ಥೆಗಳು ಕಾರ್ಯನಿರ್ವಹಿಸುವ ರೀತಿಯನ್ನು ಕೈಗೆ ಹೆಗಲು ತೋರಿಸುವ ಉದಾಹರಣೆ
- **ಉತ್ತಮ ಅಭ್ಯಾಸಗಳು**: ನೈತಿಕ ಏಐ ಅಭಿವೃದ್ಧಿ ಮತ್ತು ನಿಯೋಜನೆಗೆ ಅಗತ್ಯ ಮಾರ್ಗದರ್ಶಕ ತತ್ವಗಳು
- **[→ ಅಧ್ಯಾಯ 5 ಪ್ರಾರಂಭಿಸಿ](./05-ResponsibleGenAI/README.md)**
