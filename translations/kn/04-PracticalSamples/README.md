# ಪ್ರಾಯೋಗಿಕ ಅನ್ವಯಿಕೆಗಳು ಮತ್ತು ಯೋಜನೆಗಳು

## ನೀವು ಏನು ಕಲಿಯುತ್ತೀರಿ
ಈ ವಿಭಾಗದಲ್ಲಿ ನಾವು ಜಾವಾ ಸಹಿತ ಜನರೇಟಿವ್ AI ಅಭಿವೃದ್ಧಿ ಮಾದರಿಗಳನ್ನು ತೋರಿಸುವ ಮೂರು ಪ್ರಾಯೋಗಿಕ ಅನ್ವಯಿಕೆಗಳನ್ನು ಡೆಮೋ ಮಾಡುತ್ತೇವೆ:
- ಕ್ಲೈಂಟ್-ಸೈಡ್ ಮತ್ತು ಸರ್ವರ್-ಸೈಡ್ AI ಅನ್ನು ಸಂಯೋಜಿಸುವ ಬಹು-ಮಾಡಲ್ ಪೆಟ್ ಸ್ಟೋರಿ ಜನರೇಟರ್ ರಚಿಸಿ
- Foundry Local Spring Boot ಡೆಮೋದಿಂದ ಸ್ಥಳೀಯ AI ಮಾದರಿ ಒಕ್ಕೂಟವನ್ನು ಜಾರಿಗೆ ತಂದಿರಿ
- ಕ್ಯಾಲ್ಕ್ಯುಲೇಟರ್ ಉದಾಹರಣೆಯೊಂದಿಗೆ ಮಾದರಿ ಸಂಧರ್ಭ ಪ್ರೋಟೋಕಾಲ್ (MCP) ಸೇವೆಯನ್ನು ಅಭಿವೃದ್ಧಿಪಡಿಸಿ

## ಪರಿಚಯ

ಈ ಅಧ್ಯಾಯವು **ನಮೂನಾ ಯೋಜನೆಗಳನ್ನು** ತೋರಿಸುತ್ತದೆ, ಅವು ಜಾವಾ ಸಹಿತ ಜನರೇಟಿವ್ AI ಅಭಿವೃದ್ಧಿ ಮಾದರಿಗಳನ್ನು ಪ್ರದರ್ಶಿಸುತ್ತವೆ. ಪ್ರತಿ ಯೋಜನೆಯು ಪೂರ್ಣ ಕಾರ್ಯಕ್ಷಮವಾಗಿದೆ ಮತ್ತು ನೀವು ನಿಮ್ಮದೇ ಅನ್ವಯಿಕೆಗಳಿಗೆ ಹೊಂದಿಕೊಳ್ಳಬಹುದಾದ ವಿಶೇಷ AI ತಂತ್ರಜ್ಞಾನಗಳು, ವಾಸ್ತುಶಿಲ್ಪ ಮಾದರಿಗಳು ಮತ್ತು ಬೆಸ್ಟ್ ಪ್ರಾಕ್ಟೀಸ್‌ಗಳನ್ನು ತೋರಿಸುತ್ತದೆ.

### Foundry Local Spring Boot ಡೆಮೋ

**[Foundry Local Spring Boot ಡೆಮೋ](foundrylocal/README.md)** ಸ್ಥಳೀಯ AI ಮಾದರಿಗಳೊಂದಿಗೆ **OpenAI Java SDK** ಯೊಡನೆ ಸಂಯೋಜಿಸುವ ಮಾರ್ಗವನ್ನು ತೋರಿಸುತ್ತದೆ. ಇದು Foundry Local ನಲ್ಲಿ ನಡೆಯುತ್ತಿರುವ ಮಾದರಿಗಳೊಂದಿಗೆ (ఉದಾಹರಣೆకి, **Phi-4-mini**) ಸಂಪರ್ಕವನ್ನು ತೋರಿಸುತ್ತದೆ, ಸ್ವಯಂಚಾಲಿತ ಮಾದರಿ ಗುರುತಿಸುವಿಕೆಯನ್ನು ಒಳಗೊಂಡಿದ್ದು, ನಿಮಗೆ ಕ್ಲೌಡ್ ಸೇವೆಗಳ ಮೇಲೆ ಅವಲಂಬಿಸದೇ AI ಅನ್ವಯಿಕೆಗಳನ್ನು ಚಲಿಸಲು ಅವಕಾಶ ನೀಡುತ್ತದೆ.

### ಪೆಟ್ ಸ್ಟೋರಿ ಜನರೇಟರ್

**[ಪೆಟ್ ಸ್ಟೋರಿ ಜನರೇಟರ್](petstory/README.md)** ಒಂದು ಆಕರ್ಷಕ Spring Boot ವೆಬ್ ಅನ್ವಯಿಕೆ ಆಗಿದ್ದು, ಸೃಜನಾತ್ಮಕ ಪೆಟ್ ಕಥಾನಕಗಳನ್ನು ಉತ್ಪಾದಿಸಲು **ಬಹು-ಮಾಡಲ್ AI ಪ್ರಕ್ರಿಯೆ** ತೋರಿಸುತ್ತದೆ. ಇದರಲ್ಲಿ ಬ್ರೌಸರ್ ಆಧಾರಿತ AI ಸಂವಹನಗಳಿಗೆ transformer.js ಅನ್ನು ಮತ್ತು ಸರ್ವರ್-ಸೈಡ್ ಪ್ರಕ್ರಿಯೆಗೆ OpenAI SDK ಯನ್ನು ಉಪಯೋಗಿಸಿ ಕ್ಲೈಂಟ್-ಸೈಡ್ ಮತ್ತು ಸರ್ವರ್-ಸೈಡ್ AI ಸಾಮರ್ಥ್ಯಗಳು ಸಂಯೋಜಿಸಲ್ಪಟ್ಟಿವೆ.

### MCP ಕ್ಯಾಲ್ಕ್ಯುಲೇಟರ್ ಸೇವೆ (ಆರಂಭಿಕ ಸ್ನೇಹಿ MCP ಡೆಮೋ)

**[MCP ಕ್ಯಾಲ್ಕ್ಯುಲೇಟರ್ ಸೇವೆ](calculator/README.md)** Spring AI ಬಳಸಿ **ಮಾದರಿ ಸಂಧರ್ಭ ಪ್ರೋಟೋಕಾಲ್ (MCP)** ಯ ಸರಳ ಪ್ರದರ್ಶನ. ಇದು MCP ನಿರ್ದಿಷ್ಟ ತತ್ವಗಳಿಗೆ ಆರಂಭಿಕ ಪರಿಚಯ ನೀಡುತ್ತದೆ, MCP ಕ್ಲೈಂಟ್‌ಗಳೊಂದಿಗೆ ಸಂವಹನ ಮಾಡುವ ಮೂಲ MCP ಸರ್ವರ್ ಸರಳ ರಚಿಸುವ ವಿಧಾನ ತೋರಿಸುತ್ತದೆ.

## ಕಲಿಕೆಯ ಪ್ರಗತಿ

ಈ ಯೋಜನೆಗಳು ಹಿಂದಿನ ಅಧ್ಯಾಯಗಳಲ್ಲಿನ ಕಲಿಕೆ ತತ್ತ್ವಗಳ ಮೇರೆಗೆ ರೂಪಾಂತರವಾಗಿವೆ:

1. **ಸರಳತೆಯಿಂದ ಪ್ರಾರಂಭಿಸಿ**: Foundry Local Spring Boot ಡೆಮೋದಿಂದ ಸ್ಥಳೀಯ ಮಾದರಿಗಳೊಂದಿಗೆ ಮೂಲ AI ಸಂಯೋಜನೆಯನ್ನು ಅರಿತುಕೊಳ್ಳಿ
2. **ಪರಸ್ಪರ ಕ್ರಿಯೆ ಸೇರಿಸಿ**: ಬಹು-ಮಾಡಲ್ AI ಮತ್ತು ವೆಬ್ ಆಧಾರಿತ ಸಂವಹನಗಳಿಗೆ ಪೆಟ್ ಸ್ಟೋರಿ ಜನರೇಟರ್ ಕಡೆಗೆ ಮುಂದುವರಿಯಿರಿ
3. **MCP ಮೂಲಗಳನ್ನು ಕಲಿಯಿರಿ**: ಮಾದರಿ ಸಂಧರ್ಭ ಪ್ರೋಟೋಕಾಲ್ ಮೂಲಭೂತಗಳನ್ನು ತಿಳಿಯಲು MCP ಕ್ಯಾಲ್ಕ್ಯುಲೇಟರ್ ಸೇವೆಯನ್ನು ಪ್ರಯತ್ನಿಸಿ

## ಸಾರಾಂಶ

ಅದ್ಭುತ ಕೆಲಸ! ನೀವು ಈಗ ಕೆಲ ನಿಜವಾದ ಅನ್ವಯಿಕೆಗಳನ್ನು ಅನ್ವೇಷಿಸಿದ್ದೀರಿ:

- ಬ್ರೌಸರ್ ಮತ್ತು ಸರ್ವರ್ ಎರಡರಲ್ಲೂ ಕಾರ್ಯನಿರ್ವಹಿಸುವ ಬಹು-ಮಾಡಲ್ AI ಅನುಭವಗಳು
- ಆಧುನಿಕ ಜಾವಾ ಫ್ರೇಮ್‌ವರ್ಕ್‌ಗಳು ಮತ್ತು SDK ಗಳೊಂದಿಗೆ ಸ್ಥಳೀಯ AI ಮಾದರಿ ಒಕ್ಕೂಟ
- AI ಒಕ್ಕೂಟಗಳಿಗೆ ಉಪಕರಣಗಳನ್ನು ಸಂಯೋಜಿಸುವುದನ್ನು ನೋಡಿ ನಿಮ್ಮ ಮೊದಲ ಮಾದರಿ ಸಂಧರ್ಭ ಪ್ರೋಟೋಕಾಲ್ ಸೇವೆ
