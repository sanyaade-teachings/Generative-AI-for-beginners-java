# व्यावहारिक अनुप्रयोग आणि प्रकल्प

## आपण काय शिकाल
या विभागात आपण Java सह जनरेटिव्ह AI विकास पद्धती दर्शविणारे तीन व्यावहारिक अनुप्रयोग डेमो करू:
- क्लायंट-साइड आणि सर्व्हर-साइड AI एकत्र करून मल्टी-मोडल पाळीव प्राणी कथा जनरेटर तयार करा
- Foundry Local Spring Boot डेमो सह स्थानिक AI मॉडेल इंटिग्रेशन अंमलात आणा
- Calculator उदाहरणासह Model Context Protocol (MCP) सेवा विकसित करा

## परिचय

हा अध्याय Java सह जनरेटिव्ह AI विकास पद्धती दर्शविणारे **नमुना प्रकल्प** सादर करतो. प्रत्येक प्रकल्प पूर्णपणे कार्यशील आहे आणि विशिष्ट AI तंत्रज्ञान, आर्किटेक्चरल पद्धती, आणि सर्वोत्तम पद्धती दाखवतो ज्याला आपण आपल्या अनुप्रयोगांसाठी अनुकूल करू शकता.

### Foundry Local Spring Boot Demo

**[Foundry Local Spring Boot Demo](foundrylocal/README.md)** ही **OpenAI Java SDK** वापरून स्थानिक AI मॉडेल्ससह कसे इंटिग्रेट करायचे हे दाखवते. हे Foundry Local वर चालणाऱ्या मॉडेल्सशी कनेक्ट होण्याचे प्रदर्शन करते (उदा., **Phi-4-mini**), स्वयंचलित मॉडेल डिटेक्शनसह, ज्यामुळे आपण क्लाउड सेवा न वापरता AI अनुप्रयोग चालवू शकता.

### Pet Story Generator

**[Pet Story Generator](petstory/README.md)** हे एक आकर्षक Spring Boot वेब अनुप्रयोग आहे जे **मल्टी-मोडल AI प्रोसेसिंग** वापरून सर्जनशील पाळीव प्राणी कथा तयार करण्याचे प्रदर्शन करते. हे क्लायंट-साइड आणि सर्व्हर-साइड AI क्षमता जोडते, जिथे ब्राउझरवर आधारित AI संवादांसाठी transformer.js वापरले जाते आणि सर्व्हर-साइड प्रक्रियेसाठी OpenAI SDK वापरले जाते.

### MCP Calculator Service (Beginner-Friendly MCP Demo)

**[MCP Calculator Service](calculator/README.md)** हा Spring AI वापरून **Model Context Protocol (MCP)** चे सोपे प्रदर्शन आहे. हे MCP संकल्पनांसाठी सुरुवातीच्या स्तराचे परिचय देते, ज्यातून आपण कसे मूलभूत MCP सर्व्हर तयार करू शकता आणि ते MCP क्लायंट्सशी संवाद साधते हे दर्शवते.

## शिक्षण प्रगती

हे प्रकल्प मागील अध्यायांतील संकल्पनांवर आधारित तयार केले आहेत:

1. **सोल्या पासून सुरुवात करा**: स्थानिक मॉडेल्ससह मूलभूत AI इंटिग्रेशन समजून घेण्यासाठी Foundry Local Spring Boot Demo पासून सुरुवात करा
2. **परस्परसंवाद वाढवा**: मल्टी-मोडल AI आणि वेब-आधारित संवादांसाठी Pet Story Generator पर्यंत प्रगती करा
3. **MCP मूलभूत ज्ञान मिळवा**: Model Context Protocol ची मुळ तत्त्वे समजून घेण्यासाठी MCP Calculator Service वापरून पहा

## सारांश

छान काम! आपण आता काही वास्तविक अनुप्रयोगांचा शोध घेतला आहे:

- ब्राउझर आणि सर्व्हर दोघीकडे कार्य करणारे मल्टी-मोडल AI अनुभव
- आधुनिक Java फ्रेमवर्क आणि SDK वापरून स्थानिक AI मॉडेल इंटिग्रेशन
- साध्या Model Context Protocol सेवेसह AI सह उपकरणे कशी इंटिग्रेट करतात हे पाहिले
