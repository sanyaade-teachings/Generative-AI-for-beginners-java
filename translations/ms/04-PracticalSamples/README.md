## Pengenalan

Bab ini mempamerkan **projek contoh** yang menunjukkan corak pembangunan AI generatif dengan Java. Setiap projek berfungsi sepenuhnya dan menunjukkan teknologi AI tertentu, corak seni bina, dan amalan terbaik yang boleh anda adaptasi untuk aplikasi anda sendiri.

### Demo Foundry Local Spring Boot

**[Demo Foundry Local Spring Boot](foundrylocal/README.md)** menunjukkan cara mengintegrasi dengan model AI tempatan menggunakan **OpenAI Java SDK**. Ia mempamerkan sambungan ke model yang berjalan di Foundry Local (contoh, **Phi-4-mini**), dengan pengesanan model automatik, membolehkan anda menjalankan aplikasi AI tanpa bergantung pada perkhidmatan awan.

### Penjana Cerita Haiwan Peliharaan

**[Penjana Cerita Haiwan Peliharaan](petstory/README.md)** adalah aplikasi web Spring Boot yang menarik yang menunjukkan **pemprosesan AI multi-modal** untuk menghasilkan cerita haiwan peliharaan yang kreatif. Ia menggabungkan kebolehan AI sisi klien dan sisi pelayan menggunakan transformer.js untuk interaksi AI berasaskan pelayar dan OpenAI SDK untuk pemprosesan sisi pelayan.

### Perkhidmatan Kalkulator MCP (Demo MCP Mesra-Pemula)

**[Perkhidmatan Kalkulator MCP](calculator/README.md)** adalah demonstrasi ringkas **Protokol Konteks Model (MCP)** menggunakan Spring AI. Ia menyediakan pengenalan mesra-pemula kepada konsep MCP, menunjukkan cara mencipta MCP Server asas yang berinteraksi dengan klien MCP.

## Progres Pembelajaran

Projek ini direka untuk membina konsep dari bab sebelumnya:

1. **Mulakan Dengan Mudah**: Mula dengan Demo Foundry Local Spring Boot untuk memahami integrasi AI asas dengan model tempatan
2. **Tambah Interaktiviti**: Teruskan ke Penjana Cerita Haiwan Peliharaan untuk AI multi-modal dan interaksi berasaskan web
3. **Belajar Asas MCP**: Cuba Perkhidmatan Kalkulator MCP untuk memahami asas Protokol Konteks Model

## Ringkasan

Kerja yang bagus! Anda kini telah meneroka beberapa aplikasi sebenar:

- Pengalaman AI multi-modal yang berfungsi di pelayar dan di pelayan
- Integrasi model AI tempatan menggunakan rangka kerja dan SDK Java moden
- Perkhidmatan Protokol Konteks Model pertama anda untuk melihat bagaimana alat berintegrasi dengan AI
