# പ്രായോഗിക അപേക്ഷകളും പ്രോജക്ടുകളും

## നിങ്ങൾ പഠിക്കുന്നതു്
ഈ വകുപ്പ് മൂന്ന് പ്രായോഗിക അപേക്ഷകളെ ഡെമോ ചെയ്യുന്നു, ജാവയുമായി ജനറേറ്റീവ് എഐ ഡെവലപ്മെന്റ് പാറ്റേണുകൾ കാണിക്കുന്നവ:
- ക്ലയന്റ്-സൈഡ്, സെർവർ-സൈഡ് എഐ സംയോജിപ്പിച്ച് മൾട്ടി-മോഡൽ പെറ്റ് സ്റ്റോറി ജനറേറ്റർ സൃഷ്ടിക്കുക
- ഫൗണ്ട്രി ലോക്കൽ സ്പ്രിങ് ബൂട്ട് ഡെമോ വഴി ലോക്കൽ എഐ മോഡൽ സംയോജനം പ്രയോഗിക്കുക
- കാൽകുലേറ്റർ ഉദാഹരണത്തോടെ മോഡൽ കോൺടെക്സ്റ്റ് പ്രോട്ടോക്കോൾ (MCP) സേവനം വികസിപ്പിക്കുക

## പരിചയം

ഈ അധ്യായം ജാവയോടുള്ള ജനറേറ്റീവ് എഐ ഡെവലപ്‌മെന്റ് പാറ്റേണുകൾ കാണിക്കുന്ന **സാമ്പിൾ പ്രോജക്റ്റുകൾ** പ്രദർശിപ്പിക്കുന്നു. ഓരോ പ്രോജക്റ്റും പൂർത്തിയായ പ്രവർത്തനസജ്ജമായതിനൊപ്പം നിങ്ങൾ താങ്കളുടെ അപേക്ഷകൾക്ക് അനുയോജ്യമായ പ്രഥമകൃതികൾ, ആർക്കിടെക്ചറൽ പാറ്റേണുകൾ, മികച്ച രീതികൾ എന്നിവ കാണിക്കുന്നു.

### ഫൗണ്ട്രി ലോക്കൽ സ്പ്രിങ് ബൂട്ട് ഡെമോ

**[ഫൗണ്ട്രി ലോക്കൽ സ്പ്രിങ് ബൂട്ട് ഡെമോ](foundrylocal/README.md)** **OpenAI Java SDK** ഉപയോഗിച്ച് ലോക്കൽ എഐ മോഡലുകളുമായി എങ്ങനെ സംയോജിപ്പിക്കാമെന്നത് പ്രദർശിപ്പിക്കുന്നു. ഫൗണ്ട്രി ലോക്കലിൽ ഓടുന്ന മോഡലുകളിലേക്ക് (ഉദാഹരണത്തിന്, **Phi-4-mini**) സ്വയമേവ മോഡൽ കണ്ടെത്തലും, ക്ലൗഡ് സേവനങ്ങളിൽ ആശ്രയിക്കാതെ എഐ പ്രയോഗങ്ങൾ പ്രവർത്തിപ്പിക്കാനുമുള്ള കഴിവും കാണിക്കുന്നു.

### പെറ്റ് സ്റ്റോറി ജനറേറ്റർ

**[പെറ്റ് സ്റ്റോറി ജനറേറ്റർ](petstory/README.md)** പ്രസക്തമായ ഒരു സ്പ്രിങ് ബൂട്ട് വെബ് ആപ്ലിക്കേഷൻ ആണ്, **മൾട്ടി-മോഡൽ എഐ പ്രോസസ്സിംഗ്** ഉപയോഗിച്ച് സൃഷ്ടിപരമായ പെറ്റ് കഥകൾ നിർമ്മിക്കുന്നു. ബ്രൗസർ ആധാരിത എഐ ഇടപെടലുകൾക്കായി transformer.js ഉപയോഗിച്ച് ക്ലയന്റ്-സൈഡ്, സെർവർ-പക്ഷ എഐ പ്രോസസ്സിംഗിന് OpenAI SDK സംയോജിപ്പിച്ചിരിക്കുന്നു.

### MCP കാൽകുലേറ്റർ സേവനം (വിദ്യാർത്ഥിനികൾക്ക് സൗഹൃദമായ MCP ഡെമോ)

**[MCP കാൽകുലേറ്റർ സേവനം](calculator/README.md)** സ്പ്രിങ് എഐ ഉപയോഗിച്ചുള്ള **മോഡൽ കോൺടെക്സ്റ്റ് പ്രോട്ടോക്കോൾ (MCP)** ലളിതമായ ഒരു ഡെമോ ആണ്. MCP ആശയങ്ങൾക്കുള്ള വിദ്യാർത്ഥിനികൾക്ക് സൗഹൃദമായ പരിചയം നൽകുന്നു, MCP ക്ലയന്റുകളുമായി ഇടപെടുന്ന ഒരു അടിസ്ഥാന MCP സർവറെ നിർമ്മിക്കുന്നതിൻറെ ദൃശ്യമാണ്.

## പഠന പുരോഗതി

ഈ പ്രോജക്റ്റുകൾ മുമ്പത്തെ അധ്യായങ്ങളിൽ നിന്നുള്ള ആശയങ്ങളെ അടിസ്ഥാനമാക്കി രൂപകല്‍പ്പന ചെയ്തതാണ്:

1. **ലളിതമായി തുടങ്ങുക**: ലൊക്കൽ മോഡലുകളുമായി അടിസ്ഥാന എഐ സംയോജനം മനസിലാക്കാൻ ഫൗണ്ട്രി ലോക്കൽ സ്പ്രിങ് ബൂട്ട് ഡെമോ ആരംഭിക്കുക
2. **ഇടപെടൽ കൂട്ടുക**: മൾട്ടി-മോഡൽ എഐയും വെബ് ആദ്രവ്യവഹാരങ്ങളും കാണാൻ പെറ്റ് സ്റ്റോറി ജനറേറ്ററിലേക്ക് എത്തുക
3. **MCP അടിസ്ഥാനങ്ങൾ മനസിലാക്കുക**: മോഡൽ കോൺടെക്സ്റ്റ് പ്രോട്ടോക്കോൾ അടിസ്ഥാനങ്ങൾ പഠിക്കാൻ MCP കാൽകുലേറ്റർ സേവനം പരീക്ഷിക്കുക

## സംഗ്രഹം

നല്ല ജോലി! നിങ്ങൾ ഇപ്പോൾ ചില യഥാർത്ഥ അപേക്ഷകൾ പഠിച്ചു:

- ബ്രൗസറിലും സെർവറിലും പ്രവർത്തിക്കുന്ന മൾട്ടി-മോഡൽ എഐ അനുഭവങ്ങൾ
- ആധുനിക ജावा ഫ്രെയിംവർകുകളും SDK കളുമാണുള്ള ലോക്കൽ എഐ മോഡൽ സംയോജനം
- എഐയുമായി ഉപകരണങ്ങൾ എങ്ങനെ സംയോജിപ്പിക്കുന്നുവെന്ന് കാണാൻ നിങ്ങളുടെ ആദ്യ മോഡൽ കോൺടെക്സ്റ്റ് പ്രോട്ടോക്കോൾ സേവനം
