# Praktične Primjene i Projekti

## Što ćete naučiti
U ovom ćemo odjeljku demonstrirati tri praktične aplikacije koje prikazuju obrasce razvoja generativne AI s Javanom:
- Izraditi višemodalni Generator Priča o Kućnim Ljubimcima koji spaja klijentsku i serversku AI
- Implementirati lokalnu integraciju AI modela s Foundry Local Spring Boot demo verzijom
- Razviti uslugu Model Context Protocol (MCP) s primjerom Kalkulatora

## Uvod

Ovo poglavlje prikazuje **primjere projekata** koji demonstriraju obrasce razvoja generativne AI s Javanom. Svaki je projekt potpuno funkcionalan i demonstrira specifične AI tehnologije, arhitektonske uzorke i najbolje prakse koje možete prilagoditi za vlastite aplikacije.

### Foundry Local Spring Boot Demo

**[Foundry Local Spring Boot Demo](foundrylocal/README.md)** pokazuje kako integrirati lokalne AI modele koristeći **OpenAI Java SDK**. Prikazuje povezivanje s modelima koji rade na Foundry Local (npr., **Phi-4-mini**), s automatskim otkrivanjem modela, što vam omogućava pokretanje AI aplikacija bez ovisnosti o uslugama u oblaku.

### Generator Priča o Kućnim Ljubimcima

**[Generator Priča o Kućnim Ljubimcima](petstory/README.md)** je zanimljiva Spring Boot web aplikacija koja prikazuje **višemodalnu AI obradu** za stvaranje kreativnih priča o kućnim ljubimcima. Kombinira klijentske i serverske AI mogućnosti koristeći transformer.js za AI interakcije u pregledniku i OpenAI SDK za obradu na strani servera.

### MCP Usloga Kalkulatora (Demo MCP za početnike)

**[MCP Usloga Kalkulatora](calculator/README.md)** jednostavna je demonstracija **Model Context Protocola (MCP)** koristeći Spring AI. Nudi uvod prilagođen početnicima u MCP koncepte, pokazujući kako stvoriti osnovni MCP Server koji komunicira s MCP klijentima.

## Napredak u učenju

Ovi su projekti dizajnirani da nadograde koncepte iz prethodnih poglavlja:

1. **Počnite jednostavno**: Započnite s Foundry Local Spring Boot demo verzijom za razumijevanje osnovne integracije AI lokalnih modela
2. **Dodajte interaktivnost**: Nastavite s Generatorom Priča o Kućnim Ljubimcima za višemodalnu AI i web-bazirane interakcije
3. **Naučite osnove MCP-a**: Isprobajte MCP Uslugu Kalkulatora za razumijevanje osnovnih principa Model Context Protocola

## Sažetak

Dobar posao! Sada ste istražili neke stvarne aplikacije:

- Višemodalna AI iskustva koja rade i u pregledniku i na serveru
- Lokalna integracija AI modela koristeći moderne Java okvire i SDK-ove
- Vašu prvu MCP uslugu za razumijevanje kako alati integriraju AI
