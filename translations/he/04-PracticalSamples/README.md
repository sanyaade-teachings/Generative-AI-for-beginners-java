- יישום אינטגרציה עם מודל AI מקומי באמצעות דמו Foundry Local Spring Boot
- פיתוח שירות פרוטוקול הקשר מודלים (MCP) עם דוגמת המחשבון

## מבוא

פרק זה מציג **פרויקטים לדוגמה** שמדגימים דפוסי פיתוח בינה מלאכותית יצירתית עם Java. כל פרויקט הוא פונקציונלי במלואו ומדגים טכנולוגיות AI ספציפיות, דפוסי ארכיטקטורה ונהלים מומלצים שתוכלו להתאים ליישומים שלכם.

### דמו Foundry Local Spring Boot

ה-**[דמו Foundry Local Spring Boot](foundrylocal/README.md)** מדגים כיצד לשלב עם מודלים מקומיים של AI באמצעות **OpenAI Java SDK**. הוא מציג חיבור למודלים שרצים ב-Foundry Local (למשל, **Phi-4-mini**), עם זיהוי אוטומטי של מודלים, המאפשר הפעלת יישומי AI ללא תלות בשירותי ענן.

### מחולל סיפורי חיות מחמד

ה-**[מחולל סיפורי חיות מחמד](petstory/README.md)** הוא יישום רשת Spring Boot מרתק שמדגים **עיבוד AI ריבו-מודלי** ליצירת סיפורי חיות מחמד יצירתיים. הוא משלב יכולות AI בצד הלקוח והשרת באמצעות transformer.js לאינטראקציות מבוססות דפדפן ו-OpenAI SDK לעיבוד בצד השרת.

### שירות מחשבון MCP (דמו MCP למתחילים)

ה-**[שירות מחשבון MCP](calculator/README.md)** הוא הדגמה פשוטה של **פרוטוקול הקשר מודלים (MCP)** באמצעות Spring AI. הוא מספק היכרות למתחילים עם מושגי MCP, ומראה כיצד ליצור שרת MCP בסיסי שמתקשר עם לקוחות MCP.

## התקדמות הלמידה

הפרויקטים האלה מעוצבים לבנות על מושגים מהפרקים הקודמים:

1. **התחל בפשוט**: התחל עם דמו Foundry Local Spring Boot להבנת אינטגרציה בסיסית של AI עם מודלים מקומיים  
2. **הוסף אינטראקטיביות**: המשך למחולל סיפורי חיות מחמד עבור AI ריבו-מודלי ואינטראקציות מבוססות רשת  
3. **למד יסודות MCP**: נסה את שירות מחשבון MCP להבנת יסודות פרוטוקול הקשר מודלים

## סיכום

עבודה טובה! עתה חקרת כמה יישומים אמיתיים:

- חוויות AI ריבו-מודליות שפועלות גם בדפדפן וגם בשרת  
- אינטגרציה עם מודלים מקומיים של AI באמצעות מסגרות ו-SDK מודרניים ב-Java  
- שירות פרוטוקול הקשר מודלים ראשון שיאפשר לך לראות כיצד כלים משתלבים עם AI
