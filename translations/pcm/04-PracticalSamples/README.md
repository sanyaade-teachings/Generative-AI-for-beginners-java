## Introduction

Dis chapter dey show **sample projects** wey demonstrate generative AI development patterns wit Java. Each project dey fully functional and dey show specific AI technologies, architectural patterns, and beta practices wey you fit adapt for your own applications.

### Foundry Local Spring Boot Demo

Di **[Foundry Local Spring Boot Demo](foundrylocal/README.md)** dey show how to integrate wit local AI models using di **OpenAI Java SDK**. E dey show how to connect to models wey dey run for Foundry Local (e.g., **Phi-4-mini**), wit automatic model detection, wey allow you run AI applications without to depend on cloud services.

### Pet Story Generator

Di **[Pet Story Generator](petstory/README.md)** na one better Spring Boot web application wey show **multi-modal AI processing** to generate creative pet stories. E combine client-side and server-side AI capabilities using transformer.js for browser-based AI interactions and di OpenAI SDK for server-side processing.

### MCP Calculator Service (Beginner-Friendly MCP Demo)

Di **[MCP Calculator Service](calculator/README.md)** na one simple demonstration of di **Model Context Protocol (MCP)** using Spring AI. E provide beginner-friendly introduction to MCP concepts, wey dey show how to create basic MCP Server wey fit interact wit MCP clients.

## Learning Progression

These projects dem design to build upon concepts from previous chapters:

1. **Start Simple**: Begin wit di Foundry Local Spring Boot Demo to understand basic AI integration wit local models
2. **Add Interactivity**: Progress to di Pet Story Generator for multi-modal AI and web-based interactions
3. **Learn MCP Basics**: Try di MCP Calculator Service to understand Model Context Protocol fundamentals
