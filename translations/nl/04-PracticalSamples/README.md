## Introductie

Dit hoofdstuk toont **voorbeeldprojecten** die ontwikkelpatronen van generatieve AI met Java demonstreren. Elk project is volledig functioneel en laat specifieke AI-technologieën, architectuurpatronen en best practices zien die je kunt aanpassen voor je eigen toepassingen.

### Foundry Local Spring Boot Demo

De **[Foundry Local Spring Boot Demo](foundrylocal/README.md)** toont hoe je integreert met lokale AI-modellen met de **OpenAI Java SDK**. Het demonstreert verbindingen met modellen die draaien op Foundry Local (bijvoorbeeld **Phi-4-mini**), met automatische modeldetectie, waardoor je AI-toepassingen kunt draaien zonder afhankelijk te zijn van cloudservices.

### Huisdier Verhalen Generator

De **[Huisdier Verhalen Generator](petstory/README.md)** is een boeiende Spring Boot webapplicatie die **multimodale AI-verwerking** demonstreert om creatieve huisdierverhalen te genereren. Het combineert client-side en server-side AI-mogelijkheden met transformer.js voor AI-interacties in de browser en de OpenAI SDK voor server-side verwerking.

### MCP Calculator Service (Beginner-vriendelijke MCP Demo)

De **[MCP Calculator Service](calculator/README.md)** is een eenvoudige demonstratie van het **Model Context Protocol (MCP)** met Spring AI. Het biedt een beginner-vriendelijke introductie tot MCP-concepten en laat zien hoe je een basis MCP Server maakt die interactie heeft met MCP-clients.

## Leerprogressie

Deze projecten zijn ontworpen om voort te bouwen op concepten uit eerdere hoofdstukken:

1. **Begin eenvoudig**: Start met de Foundry Local Spring Boot Demo om de basis van AI-integratie met lokale modellen te begrijpen
2. **Voeg interactiviteit toe**: Ga verder met de Huisdier Verhalen Generator voor multimodale AI en webgebaseerde interacties
3. **Leer MCP basisprincipes**: Probeer de MCP Calculator Service om de basisprincipes van het Model Context Protocol te begrijpen

## Samenvatting

Goed gedaan! Je hebt nu een aantal echte toepassingen verkend:

- Multimodale AI-ervaringen die zowel in de browser als op de server werken
