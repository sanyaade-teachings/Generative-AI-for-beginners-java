# کاربردها و پروژه‌های عملی

## آنچه یاد خواهید گرفت
در این بخش سه برنامه کاربردی عملی را نمایش خواهیم داد که الگوهای توسعه هوش مصنوعی مولد با جاوا را نشان می‌دهند:
- ساخت یک مولد داستان حیوان خانگی چندرسانه‌ای که ترکیبی از هوش مصنوعی سمت کلاینت و سمت سرور است
- پیاده‌سازی ادغام مدل هوش مصنوعی محلی با نمونه نمایشی Foundry Local Spring Boot
- توسعه یک سرویس پروتکل زمینه مدل (MCP) با مثال ماشین‌حساب

## مقدمه

این فصل **نمونه پروژه‌هایی** را ارائه می‌دهد که الگوهای توسعه هوش مصنوعی مولد با جاوا را نشان می‌دهند. هر پروژه کاملاً کاربردی است و فناوری‌های خاص هوش مصنوعی، الگوهای معماری و بهترین شیوه‌ها را که می‌توانید برای برنامه‌های خود تطبیق دهید، نشان می‌دهد.

### نمونه نمایشی Foundry Local Spring Boot

**[نمونه نمایشی Foundry Local Spring Boot](foundrylocal/README.md)** نحوه ادغام با مدل‌های هوش مصنوعی محلی با استفاده از **OpenAI Java SDK** را نشان می‌دهد. این برنامه اتصال به مدل‌های اجرا شده روی Foundry Local (مثلاً **Phi-4-mini**) را با تشخیص خودکار مدل نشان می‌دهد، که به شما اجازه می‌دهد برنامه‌های هوش مصنوعی را بدون وابستگی به خدمات ابری اجرا کنید.

### مولد داستان حیوان خانگی

**[مولد داستان حیوان خانگی](petstory/README.md)** یک برنامه وب جذاب Spring Boot است که **پردازش هوش مصنوعی چندرسانه‌ای** برای تولید داستان‌های خلاقانه حیوانات خانگی را نشان می‌دهد. این برنامه قابلیت‌های هوش مصنوعی سمت کلاینت و سمت سرور را ترکیب می‌کند و با استفاده از transformer.js برای تعاملات هوش مصنوعی در مرورگر و OpenAI SDK برای پردازش سمت سرور عمل می‌کند.

### سرویس ماشین‌حساب MCP (نمونه نمایشی ساده MCP)

**[سرویس ماشین‌حساب MCP](calculator/README.md)** نمایشی ساده از **پروتکل زمینه مدل (MCP)** با استفاده از Spring AI است. این برنامه یک معرفی ساده به مفاهیم MCP ارائه می‌کند و نشان می‌دهد چگونه یک سرور MCP ساده بسازید که با کلاینت‌های MCP تعامل دارد.

## روند یادگیری

این پروژه‌ها به گونه‌ای طراحی شده‌اند که مباحث فصول قبلی را توسعه دهند:

1. **شروع ساده**: با نمونه نمایشی Foundry Local Spring Boot شروع کنید تا ادغام پایه‌ای هوش مصنوعی با مدل‌های محلی را درک کنید
2. **افزودن تعامل**: به سمت مولد داستان حیوان خانگی برای هوش مصنوعی چندرسانه‌ای و تعاملات وب پیش بروید
3. **آموختن اصول MCP**: سرویس ماشین‌حساب MCP را امتحان کنید تا اصول پروتکل زمینه مدل را بیاموزید

## خلاصه

کار خوبی انجام دادید! اکنون برخی از برنامه‌های واقعی را بررسی کرده‌اید:

- تجربیات هوش مصنوعی چندرسانه‌ای که هم در مرورگر و هم در سرور کار می‌کنند
- ادغام مدل هوش مصنوعی محلی با استفاده از چارچوب‌ها و SDKهای مدرن جاوا
- اولین سرویس پروتکل زمینه مدل خود را دیدید تا ببینید ابزارها چگونه با هوش مصنوعی ادغام می‌شوند
