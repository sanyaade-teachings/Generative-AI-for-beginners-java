# Практичне апликације и пројекти

## Шта ћете научити

- Креирајте мулти-модални Генератор прича о кућним љубимцима који комбинује AI на клијентској и серверској страни
- Имплементирајте локалну интеграцију AI модела са Foundry Local Spring Boot демонстрацијом
- Развијте сервис за Протокол контекста модела (MCP) са примером калкулатора

## Увод

Ово поглавље представља **примерке пројеката** који показују обрасце развоја генеративне AI са Јава. Сваки пројекат је у потпуности функционалан и демонстрира специфичне AI технологије, архитектонске обрасце и добре праксе које можете прилагодити за своје апликације.

### Foundry Local Spring Boot демонстрација

**[Foundry Local Spring Boot демонстрација](foundrylocal/README.md)** показује како да се интегришете са локалним AI моделима користећи **OpenAI Java SDK**. Приказује повезивање са моделима који раде на Foundry Local (нпр. **Phi-4-mini**), са аутоматском детекцијом модела, омогућавајући вам да покрећете AI апликације без ослањања на облачне сервисе.

### Генератор прича о кућним љубимцима

**[Генератор прича о кућним љубимцима](petstory/README.md)** је занимљива веб апликација направљена помоћу Spring Boot која демонстрира **мулти-модално AI процесирање** за генерисање креативних прича о кућним љубимцима. Комбинује могућности AI са клијентске и серверске стране користећи transformer.js за AI интеракције у прегледачу и OpenAI SDK за обраду на серверској страни.

### MCP Калкулатор сервис (почетнички прилагодјена MCP демонстрација)

**[MCP Калкулатор сервис](calculator/README.md)** је једноставна демонстрација **Протокола контекста модела (MCP)** користећи Spring AI. Пружа почетнички прилагођен увод у MCP концепте, приказујући како направити основни MCP сервер који комуницира са MCP клијентима.

## Прогрес у учењу

Ови пројекти су дизајнирани да се надовезују на концепте из претходних поглавља:

1. **Почните једноставно**: Започните са Foundry Local Spring Boot демонстрацијом да разумете основну интеграцију AI са локалним моделима
2. **Додајте интерактивност**: Напредујте ка Генератору прича о кућним љубимцима за мулти-модално AI и интеракције базиране на вебу
3. **Упознајте основе MCP**: Испробајте MCP Калкулатор сервис да бисте разумели основе Протокола контекста модела

## Резиме

Добар посао! Сада сте истражили неке реалне апликације:

- Мулти-модалне AI искуства која раде и у прегледачу и на серверу
- Локалну интеграцију AI модела користећи модерне Јава оквире и SDK-ове
- Ваш први сервис Протокола контекста модела да видите како алати интегришу AI
