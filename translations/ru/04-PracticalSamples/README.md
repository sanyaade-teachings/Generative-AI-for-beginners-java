## Введение

В этой главе представлены **примерные проекты**, демонстрирующие паттерны разработки генеративного ИИ на Java. Каждый проект полностью функционален и демонстрирует конкретные технологии ИИ, архитектурные паттерны и лучшие практики, которые вы можете адаптировать для своих приложений.

### Демо Foundry Local Spring Boot

**[Foundry Local Spring Boot Demo](foundrylocal/README.md)** демонстрирует, как интегрироваться с локальными моделями ИИ, используя **OpenAI Java SDK**. Здесь показано подключение к моделям, работающим на Foundry Local (например, **Phi-4-mini**), с автоматическим обнаружением моделей, что позволяет запускать ИИ-приложения без зависимости от облачных сервисов.

### Генератор историй про питомцев

**[Pet Story Generator](petstory/README.md)** — это увлекательное веб-приложение на Spring Boot, демонстрирующее **мультимодальную обработку ИИ** для создания креативных историй про питомцев. Оно сочетает возможности клиентского и серверного ИИ, используя transformer.js для браузерного взаимодействия с ИИ и OpenAI SDK для серверной обработки.

### MCP калькулятор (демо MCP для начинающих)

**[MCP Calculator Service](calculator/README.md)** — это простой демонстрационный пример **Model Context Protocol (MCP)** с использованием Spring AI. Он предоставляет дружелюбное для начинающих введение в концепции MCP, показывая, как создать базовый сервер MCP, который взаимодействует с клиентами MCP.

## Прогресс обучения

Эти проекты созданы для постепенного изучения концепций из предыдущих глав:

1. **Начинайте с простого**: начните с демо Foundry Local Spring Boot, чтобы понять базовую интеграцию ИИ с локальными моделями
2. **Добавьте интерактивность**: перейдите к генератору историй про питомцев для мультимодального ИИ и веб-взаимодействий
3. **Изучите основы MCP**: попробуйте MCP калькулятор, чтобы понять фундаментальные принципы Model Context Protocol

## Итог

Отличная работа! Теперь вы изучили несколько реальных приложений:

- Мультимодальные ИИ-сценарии, работающие как в браузере, так и на сервере
