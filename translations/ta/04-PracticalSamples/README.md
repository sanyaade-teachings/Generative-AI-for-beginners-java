# நடைமுறை பயன்பாடுகள் மற்றும் திட்டங்கள்

## நீங்கள் கற்றுக்கொள்ளவிருக்கும் விஷயங்கள்
இந்த பிரிவில் நாமும் ஜாவாவுடன் ஜெனரேட்டிவ் AI மேம்பாட்டு மாதிரிகளை வெளிப்படுத்தும் மூன்று நடைமுறை பயன்பாடுகளை மதிப்பீடு செய்வோம்:
- கிளையன்ட் பக்கம் மற்றும் சர்வர் பக்கம் AI-யை ஒருங்கிணைத்து பலவகை கொண்ட Pet Story Generator உருவாக்குதல்
- Foundry Local Spring Boot டெமோவுடன் உள்ளூர் AI மாதிரி ஒருங்கிணைப்பு செயல்படுத்துதல்
- Calculator உதாரணத்துடன் Model Context Protocol (MCP) சேவையை மேம்படுத்துதல்

## அறிமுகம்

இந்த அத்தியாயம் ஜாவாவுடன் ஜெனரேட்டிவ் AI மேம்பாட்டு மாதிரிகளை வெளிப்படுத்தும் **மாதிரி திட்டங்களை** காட்டுகிறது. ஒவ்வொரு திட்டமும் முழுமையாக செயல்திறன் வாய்ந்தது மற்றும் குறிப்பிட்ட AI தொழில்நுட்பங்கள், கட்டமைப்பு மாதிரிகள் மற்றும் சிறந்த நடைமுறைகளை நீங்கள் உங்கள் சொந்த பயன்பாடுகளுக்கு ஏற்ப மாற்றிக்கொள்ளலாம்.

### Foundry Local Spring Boot டெமோ

**[Foundry Local Spring Boot டெமோ](foundrylocal/README.md)**, **OpenAI Java SDK** மூலம் உள்ளூர் AI மாதிரிகளுடன் ஒருங்கிணைப்பதை எவ்வாறு செய்வது என்பதைக் காட்சியிடுகிறது. இது Foundry Local-ல் இயங்கும் மாதிரிகளுடன் (உதா., **Phi-4-mini**) தொடர்பு கொள்ள காட்டுகிறது, தானாக மாதிரி கண்டறிதல் செய்து, மேக சேவைகளில் சாராது AI பயன்பாடுகளை இயக்கு வாய்ப்பை அளிக்கிறது.

### Pet Story Generator

**[Pet Story Generator](petstory/README.md)** என்பது, பலவகை AI செயலாக்கத்தை வெளிப்படுத்தும் ஒரு ஈர்க்கக்கூடிய Spring Boot வலை பயன்பாடு. இது கிளையன்ட் பக்கம் மற்றும் சர்வர் பக்கம் AI திறன்களை இணைத்து, உலாவியில் இயங்கும் AI தொடர்பிற்காக transformer.js மற்றும் சர்வர் பக்கம் செயலாக்கத்திற்காக OpenAI SDK ஆகியவற்றைப் பயன்படுத்துகிறது.

### MCP Calculator சேவை (ஆரம்ப நிலையான MCP டெமோ)

**[MCP Calculator சேவை](calculator/README.md)** என்பது Spring AI கொண்டு **Model Context Protocol (MCP)** ஐ எளிய முறையில் கற்றுத்தரும் ஒரு சாதாரண காட்டும் செயலி. இது MCP கருத்துகளை ஆரம்ப நிலை பயனாளர்களுக்கு அறிமுகப்படுத்துகிறது, அடிப்படையான MCP சர்வரை உருவாக்கி MCP கிளையன்ட்களுடன் தொடர்பு கொள்ள காட்டுகிறது.

## கற்றல் முன்னேற்றம்

இந்த திட்டங்கள் முந்தைய அத்தியாயங்களில் கற்றுணர்வுகளை விரிவுபடுத்த வடிவமைக்கப்பட்டுள்ளன:

1. **எளிதில் துவங்குக**: உள்ளூர் மாதிரிகளுடன் அடிப்படையான AI ஒருங்கிணைப்பை புரிந்துகொள்ள Foundry Local Spring Boot டெமோவுடன் துவங்குக
2. **இணைந்த செயல்திறன் சேர்க்கவும்**: Pet Story Generator கொண்டு பலவகை AI மற்றும் வலை அடிப்படையிலான தொடர்பு கற்றுக்கொள்க
3. **MCP அடிப்படைகளை கற்றுக்கொள்ளவும்**: MCP Calculator சேவையை முயற்சித்து Model Context Protocol அடிப்படைகளை புரிந்துகொள்ளவும்

## சுருக்கம்

அற்புதமாக! நீங்கள் சில உண்மையான பயன்பாடுகளை ஆராய்ந்துள்ளீர்கள்:

- உலாவி மற்றும் சர்வரில் இரண்டும் செயல்படும் பலவகை AI அனுபவங்கள்
- நவீன Java கட்டமைப்புகள் மற்றும் SDK-களைப் பயன்படுத்தி உள்ளூர் AI மாதிரி ஒருங்கிணைப்பு
- உங்கள் முதலாவது Model Context Protocol சேவை மூலம் கருவிகள் AI உடன் எவ்வாறு இணைகின்றன என்பதை காண்க
