## 課程結構與學習路徑

### **第一章：生成式 AI 簡介**
- <strong>核心概念</strong>：理解大型語言模型（LLM）、tokens、向量嵌入與 AI 能力
- **Java AI 生態系統**：Spring AI 與 OpenAI SDK 簡介
- <strong>模型上下文協議</strong>：MCP 及其在 AI 代理通信中的作用介紹
- <strong>實際應用</strong>：聊天機器人與內容生成等現實場景
- **[→ 開始第一章](./01-IntroToGenAI/README.md)**

### **第二章：開發環境設置**
- **Azure AI Foundry**：用 Bicep 和 Azure Developer CLI（azd）以代碼方式配置模型部署
- **Spring Boot + Spring AI**：企業級 AI 應用開發最佳實踐
- <strong>無鑰匙認證</strong>：使用 Microsoft Entra ID 安全連接—無需管理 API 金鑰
- <strong>開發工具</strong>：Docker 容器、VS Code 與 GitHub Codespaces 配置
- **[→ 開始第二章](./02-SetupDevEnvironment/README.md)**

### **第三章：核心生成式 AI 技術**
- <strong>提示工程</strong>：優化 AI 模型回應的技術
- <strong>向量嵌入及運算</strong>：實現語意搜索與相似度匹配
- **檢索增強生成（RAG）**：結合 AI 與自有數據來源
- <strong>函數調用</strong>：以自訂工具與插件擴展 AI 功能
- **[→ 開始第三章](./03-CoreGenerativeAITechniques/README.md)**

### **第四章：實際應用與專案**
- <strong>寵物故事生成器</strong> (`petstory/`)：使用 Azure AI Foundry 創造性內容生成
- **Foundry 本地示範** (`foundrylocal/`)：本地 AI 模型與 OpenAI Java SDK 整合
- **MCP 計算器服務** (`calculator/`)：使用 Spring AI 實現基本模型上下文協議
- **[→ 開始第四章](./04-PracticalSamples/README.md)**

### **第五章：負責任的 AI 開發**
- **Azure AI Foundry 內容安全**：測試內建內容過濾與安全機制（硬性封鎖與軟拒絕）
- **負責任的 AI 示範**：實用範例展示現代 AI 安全系統運作
- <strong>最佳實踐</strong>：道德 AI 開發與部署的必要指南
- **[→ 開始第五章](./05-ResponsibleGenAI/README.md)**
