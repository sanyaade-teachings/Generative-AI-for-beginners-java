# ব্যবহারিক আবেদনসমূহ ও প্রকল্পসমূহ

## আপনি যা শিখবেন
এই বিভাগে আমরা তিনটি ব্যবহারিক অ্যাপ্লিকেশন ডেমো করব যা Java সহ জেনারেটিভ AI ডেভেলপমেন্ট প্যাটার্নগুলি প্রদর্শন করে:
- ক্লায়েন্ট-সাইড এবং সার্ভার-সাইড AI সমন্বিত একটি মাল্টি-মোডাল পেট স্টোরি জেনারেটর তৈরি করুন
- Foundry Local Spring Boot ডেমোর সাথে স্থানীয় AI মডেলের ইন্টিগ্রেশন বাস্তবায়ন করুন
- ক্যালকুলেটর উদাহরণের মাধ্যমে একটি মডেল কনটেক্সট প্রোটোকল (MCP) পরিষেবা উন্নয়ন করুন

## পরিচিতি

এই অধ্যায়টি **উদাহরণ প্রকল্পসমূহ** প্রদর্শন করে যা Java সহ জেনারেটিভ AI ডেভেলপমেন্ট প্যাটার্নগুলি উপস্থাপন করে। প্রতিটি প্রকল্প সম্পূর্ণ কার্যকর এবং নির্দিষ্ট AI প্রযুক্তি, স্থাপত্য প্যাটার্ন, এবং সেরা অনুশীলনসমূহ প্রদর্শন করে যা আপনি আপনার নিজস্ব অ্যাপ্লিকেশনগুলির জন্য অভিযোজিত করতে পারেন।

### Foundry Local Spring Boot ডেমো

**[Foundry Local Spring Boot ডেমো](foundrylocal/README.md)** দেখায় কীভাবে স্থানীয় AI মডেলগুলির সাথে **OpenAI Java SDK** ব্যবহার করে ইন্টিগ্রেট করা যায়। এটি Foundry Local-এ চালিত মডেলগুলির সাথে সংযোগ প্রদর্শন করে (যেমন, **Phi-4-mini**), স্বয়ংক্রিয় মডেল সনাক্তকরণসহ, যা আপনাকে ক্লাউড সার্ভিসের উপর নির্ভর না করে AI অ্যাপ্লিকেশন চালাতে দেয়।

### পেট স্টোরি জেনারেটর

**[পেট স্টোরি জেনারেটর](petstory/README.md)** একটি আকর্ষণীয় Spring Boot ওয়েব অ্যাপ্লিকেশন যা **মাল্টি-মোডাল AI প্রক্রিয়াকরণ** প্রদর্শন করে সৃজনশীল পেট গল্প তৈরি করতে। এটি ক্লায়েন্ট-সাইড এবং সার্ভার-সাইড AI দক্ষতাকে সংযুক্ত করে যেখানে ব্রাউজারভিত্তিক AI যোগাযোগের জন্য transformer.js এবং সার্ভার-সাইড প্রক্রিয়াকরণের জন্য OpenAI SDK ব্যবহার করা হয়।

### MCP ক্যালকুলেটর সার্ভিস (শিখন-সুবিধাজনক MCP ডেমো)

**[MCP ক্যালকুলেটর সার্ভিস](calculator/README.md)** Spring AI ব্যবহার করে **মডেল কনটেক্সট প্রোটোকল (MCP)** এর একটি সহজ প্রদর্শনী। এটি MCP ধারণাগুলির জন্য একটি শিখন-সুবিধাজনক পরিচিতি প্রদান করে, দেখায় কীভাবে একটি মৌলিক MCP সার্ভার তৈরি করা যায় যা MCP ক্লায়েন্টদের সাথে যোগাযোগ করে।

## শিক্ষা প্রগতি

এই প্রকল্পসমূহ পূর্ববর্তী অধ্যায়গুলোর ধারণাগুলির উপরে নির্মিত হয়েছে:

1. **সহজ থেকে শুরু করুন**: Foundry Local Spring Boot ডেমো দিয়ে স্থানীয় মডেলের সাথে মৌলিক AI ইন্টিগ্রেশন বুঝুন
2. **ইন্টারঅ্যাক্টিভিটি যোগ করুন**: মাল্টি-মোডাল AI এবং ওয়েবভিত্তিক ইন্টারঅ্যাকশন জন্য পেট স্টোরি জেনারেটরে অগ্রসর হন
3. **MCP মৌলিক শিখুন**: MCP ক্যালকুলেটর সার্ভিস চেষ্টা করুন মডেল কনটেক্সট প্রোটোকল-এর মৌলিক বিষয় বুঝতে

## সারাংশ

চমৎকার কাজ! আপনি এখন কিছু বাস্তব অ্যাপ্লিকেশন অন্বেষণ করেছেন:

- ব্রাউজার এবং সার্ভার উভয়ে কাজ করা মাল্টি-মোডাল AI অভিজ্ঞতা
- আধুনিক Java ফ্রেমওয়ার্ক এবং SDK ব্যবহার করে স্থানীয় AI মডেল ইন্টিগ্রেশন
- AI এর সাথে টুল ইন্টিগ্রেশন দেখতে প্রথম মডেল কনটেক্সট প্রোটোকল সার্ভিস
