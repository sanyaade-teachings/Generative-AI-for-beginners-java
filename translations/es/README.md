## Estructura del curso y ruta de aprendizaje

### **Capítulo 1: Introducción a la IA Generativa**
- **Conceptos clave**: Comprender los modelos de lenguaje grande, tokens, embeddings y capacidades de la IA
- **Ecosistema Java AI**: Visión general de Spring AI y SDKs de OpenAI
- **Protocolo de Contexto del Modelo**: Introducción al MCP y su papel en la comunicación de agentes IA
- **Aplicaciones prácticas**: Escenarios del mundo real, incluyendo chatbots y generación de contenido
- **[→ Iniciar Capítulo 1](./01-IntroToGenAI/README.md)**

### **Capítulo 2: Configuración del entorno de desarrollo**
- **Azure AI Foundry**: Provisión de implementaciones de modelos como código con Bicep y Azure Developer CLI (azd)
- **Spring Boot + Spring AI**: Mejores prácticas para el desarrollo de aplicaciones empresariales de IA
- **Autenticación sin clave**: Conéctate de forma segura con Microsoft Entra ID — sin claves API que gestionar
- **Herramientas de desarrollo**: Contenedores Docker, VS Code y configuración de GitHub Codespaces
- **[→ Iniciar Capítulo 2](./02-SetupDevEnvironment/README.md)**

### **Capítulo 3: Técnicas centrales de IA Generativa**
- **Ingeniería de prompts**: Técnicas para obtener respuestas óptimas del modelo IA
- **Embeddings y operaciones vectoriales**: Implementar búsqueda semántica y ajustes de similitud
- **Generación aumentada por recuperación (RAG)**: Combina IA con tus propias fuentes de datos
- **Llamado a funciones**: Extiende las capacidades de IA con herramientas y plugins personalizados
- **[→ Iniciar Capítulo 3](./03-CoreGenerativeAITechniques/README.md)**

### **Capítulo 4: Aplicaciones y proyectos prácticos**
- **Generador de historias para mascotas** (`petstory/`): Generación creativa de contenido con Azure AI Foundry
- **Demo Foundry local** (`foundrylocal/`): Integración local de modelos IA con OpenAI Java SDK
- **Servicio calculadora MCP** (`calculator/`): Implementación básica del Protocolo de Contexto del Modelo con Spring AI
- **[→ Iniciar Capítulo 4](./04-PracticalSamples/README.md)**

### **Capítulo 5: Desarrollo responsable de IA**
- **Seguridad de contenido en Azure AI Foundry**: Prueba los mecanismos integrados de filtrado y seguridad de contenido (bloqueos duros y rechazos suaves)
- **Demo de IA responsable**: Ejemplo práctico que muestra cómo funcionan los sistemas modernos de seguridad en IA
- **Mejores prácticas**: Directrices esenciales para el desarrollo y despliegue ético de IA
- **[→ Iniciar Capítulo 5](./05-ResponsibleGenAI/README.md)**
