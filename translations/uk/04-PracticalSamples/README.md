## Вступ

Цей розділ демонструє **приклади проєктів**, які ілюструють шаблони розробки генеративного ШІ на Java. Кожен проєкт є повнофункціональним і демонструє конкретні технології ШІ, архітектурні шаблони та кращі практики, які ви можете адаптувати для власних застосунків.

### Foundry Local Spring Boot Demo

**[Foundry Local Spring Boot Demo](foundrylocal/README.md)** демонструє, як інтегруватись з локальними моделями ШІ за допомогою **OpenAI Java SDK**. Він показує підключення до моделей, що працюють на Foundry Local (наприклад, **Phi-4-mini**), з автоматичним виявленням моделі, що дозволяє запускати застосунки ШІ без залежності від хмарних сервісів.

### Генератор історій про тварин

**[Генератор історій про тварин](petstory/README.md)** — це захоплюючий веб-застосунок Spring Boot, який демонструє **мультимодальну обробку ШІ** для створення креативних історій про домашніх улюбленців. Він поєднує можливості ШІ на стороні клієнта та сервера, використовуючи transformer.js для взаємодії з ШІ у браузері та OpenAI SDK для серверної обробки.

### Сервіс калькулятора MCP (демо MCP для початківців)

**[Сервіс калькулятора MCP](calculator/README.md)** — це простий приклад **Model Context Protocol (MCP)** з використанням Spring AI. Він пропонує дружнє для початківців введення в концепції MCP, показуючи, як створити базовий MCP-сервер, який взаємодіє з MCP-клієнтами.

## Послідовність навчання

Ці проєкти розроблені для поетапного вивчення концепцій з попередніх розділів:

1. **Почніть з простого**: розгляньте Foundry Local Spring Boot Demo, щоб зрозуміти базову інтеграцію ШІ з локальними моделями
2. **Додайте інтерктивність**: переходьте до Генератора історій про тварин для мультимодальних ШІ та веб-взаємодії
3. **Вивчайте основи MCP**: спробуйте Сервіс калькулятора MCP, щоб зрозуміти основи Model Context Protocol

## Підсумок

Чудова робота! Ви тепер ознайомилися з реальними застосунками:

- Мультимодальні ШІ-досвіди, що працюють і у браузері, і на сервері
- Інтеграція локальних моделей ШІ з використанням сучасних фреймворків та SDK Java
- Ваш перший сервіс Model Context Protocol для розуміння інтеграції інструментів з ШІ
