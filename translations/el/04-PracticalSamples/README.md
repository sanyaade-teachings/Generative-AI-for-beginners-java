## Εισαγωγή

Αυτό το κεφάλαιο παρουσιάζει **δείγματα έργων** που επιδεικνύουν τα πρότυπα ανάπτυξης γενετικής τεχνητής νοημοσύνης με Java. Κάθε έργο είναι πλήρως λειτουργικό και επιδεικνύει συγκεκριμένες τεχνολογίες τεχνητής νοημοσύνης, αρχιτεκτονικά πρότυπα και βέλτιστες πρακτικές που μπορείτε να προσαρμόσετε για τις δικές σας εφαρμογές.

### Foundry Local Spring Boot Demo

Το **[Foundry Local Spring Boot Demo](foundrylocal/README.md)** δείχνει πώς να ενσωματώσετε τοπικά μοντέλα τεχνητής νοημοσύνης χρησιμοποιώντας το **OpenAI Java SDK**. Δείχνει σύνδεση με μοντέλα που τρέχουν στο Foundry Local (π.χ., **Phi-4-mini**), με αυτόματη ανίχνευση μοντέλου, επιτρέποντάς σας να τρέχετε εφαρμογές τεχνητής νοημοσύνης χωρίς να βασίζεστε σε υπηρεσίες cloud.

### Γεννήτρια Ιστοριών Κατοικιδίων

Η **[Γεννήτρια Ιστοριών Κατοικιδίων](petstory/README.md)** είναι μια ελκυστική εφαρμογή web Spring Boot που επιδεικνύει **πολυτροπική επεξεργασία τεχνητής νοημοσύνης** για τη δημιουργία δημιουργικών ιστοριών κατοικιδίων. Συνδυάζει τις δυνατότητες τεχνητής νοημοσύνης πελάτη και διακομιστή χρησιμοποιώντας το transformer.js για αλληλεπίδραση τεχνητής νοημοσύνης στον περιηγητή και το OpenAI SDK για επεξεργασία στο διακομιστή.

### Υπηρεσία Υπολογιστή MCP (Demo MCP για Αρχάριους)

Η **[Υπηρεσία Υπολογιστή MCP](calculator/README.md)** είναι μια απλή επίδειξη του **Πρωτοκόλλου Πλαισίου Μοντέλου (MCP)** με χρήση του Spring AI. Παρέχει μια εισαγωγή φιλική προς αρχάριους στις έννοιες MCP, δείχνοντας πώς να δημιουργήσετε έναν βασικό MCP Server που αλληλεπιδρά με MCP πελάτες.

## Πρόοδος Μάθησης

Αυτά τα έργα σχεδιάστηκαν ώστε να οικοδομήσουν πάνω σε έννοιες προηγούμενων κεφαλαίων:

1. **Ξεκινήστε Απλά**: Ξεκινήστε με το Foundry Local Spring Boot Demo για να κατανοήσετε βασική ενσωμάτωση τεχνητής νοημοσύνης με τοπικά μοντέλα
2. **Προσθέστε Διαδραστικότητα**: Προχωρήστε στη Γεννήτρια Ιστοριών Κατοικιδίων για πολυτροπική τεχνητή νοημοσύνη και αλληλεπιδράσεις μέσω web
3. **Μάθετε τις Βασικές MCP**: Δοκιμάστε την Υπηρεσία Υπολογιστή MCP για να καταλάβετε τα θεμελιώδη του Πρωτοκόλλου Πλαισίου Μοντέλου

## Περίληψη

Καλή δουλειά! Τώρα έχετε εξερευνήσει κάποιες πραγματικές εφαρμογές:

- Πολυτροπικές εμπειρίες τεχνητής νοημοσύνης που λειτουργούν τόσο στον περιηγητή όσο και στον διακομιστή
- Τοπική ενσωμάτωση μοντέλου τεχνητής νοημοσύνης χρησιμοποιώντας σύγχρονα Java frameworks και SDKs
- Την πρώτη σας υπηρεσία Πρωτοκόλλου Πλαισίου Μοντέλου για να δείτε πώς τα εργαλεία ενσωματώνονται με την τεχνητή νοημοσύνη
