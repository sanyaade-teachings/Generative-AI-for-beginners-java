## 课程结构与学习路径

### **第 1 章：生成式 AI 介绍**
- <strong>核心概念</strong>：理解大型语言模型、token、embedding 及 AI 能力
- **Java AI 生态系统**：Spring AI 和 OpenAI SDK 概览
- <strong>模型上下文协议</strong>：介绍 MCP 及其在 AI 代理通信中的作用
- <strong>实际应用</strong>：聊天机器人和内容生成等实际场景
- **[→ 开始第 1 章](./01-IntroToGenAI/README.md)**

### **第 2 章：开发环境设置**
- **Azure AI Foundry**：通过 Bicep 和 Azure Developer CLI (azd) 以代码形式部署模型
- **Spring Boot + Spring AI**：企业级 AI 应用开发最佳实践
- <strong>无密钥认证</strong>：安全连接 Microsoft Entra ID — 无需管理 API 密钥
- <strong>开发工具</strong>：Docker 容器，VS Code 和 GitHub Codespaces 配置
- **[→ 开始第 2 章](./02-SetupDevEnvironment/README.md)**

### **第 3 章：核心生成式 AI 技术**
- **Prompt 工程**：实现 AI 模型最佳响应的技巧
- **Embedding 与向量操作**：实现语义搜索和相似度匹配
- **检索增强生成 (RAG)**：结合 AI 与自有数据源
- <strong>函数调用</strong>：使用自定义工具和插件扩展 AI 功能
- **[→ 开始第 3 章](./03-CoreGenerativeAITechniques/README.md)**

### **第 4 章：实际应用与项目**
- <strong>宠物故事生成器</strong>（`petstory/`）：使用 Azure AI Foundry 进行创意内容生成
- **Foundry 本地演示**（`foundrylocal/`）：使用 OpenAI Java SDK 本地集成 AI 模型
- **MCP 计算机服务**（`calculator/`）：基于 Spring AI 的基础模型上下文协议实现
- **[→ 开始第 4 章](./04-PracticalSamples/README.md)**

### **第 5 章：负责任的 AI 开发**
- **Azure AI Foundry 内容安全**：测试内置内容过滤和安全机制（硬性阻断和软性拒绝）
- **负责任 AI 演示**：实操示例展示现代 AI 安全系统的工作方式
- <strong>最佳实践</strong>：伦理 AI 开发与部署的重要指南
- **[→ 开始第 5 章](./05-ResponsibleGenAI/README.md)**
