# AGENTS.md

## ਪ੍ਰੋਜੈਕਟ ਦਾ ਝਲਕ

ਇਹ ਜਾਵਾ ਨਾਲ ਜਨੇਰੇਟਿਵ ਏਆਈ ਵਿਕਾਸ ਸਿੱਖਣ ਲਈ ਇੱਕ ਸਿੱਖਿਆ ਲਈ ਰਿਪੋਜ਼ਟਰੀ ਹੈ। ਇਹ ਲਾਰਜ ਲੈਂਗਵੇਜ ਮਾਡਲਜ਼ (LLMs), ਪ੍ਰਾਂਪਟ ਇੰਜਨੀਅਰਿੰਗ, ਐਮਬੈਡਿੰਗਜ਼, RAG (ਰੀਟਰੀਵਲ-ਆਗਮੈਂਟਡ ਜਨੇਰੇਸ਼ਨ), ਅਤੇ ਮਾਡਲ ਕਾਂਟੈਕਸਟ ਪ੍ਰੋਟੋਕੋਲ (MCP) ਨੂੰ ਸਮੇਤ ਇੱਕ ਵਿਸਤ੍ਰਿਤ ਹੱਥੋਂ ਕੋਰਸ ਪ੍ਰਦਾਨ ਕਰਦਾ ਹੈ।

**ਮੁੱਖ ਤਕਨਾਲੋਜੀਆਂ:**
- ਜਾਵਾ 21
- ਸਪ੍ਰਿੰਗ ਬੂਟ 3.5.x
- ਸਪ੍ਰਿੰਗ ਏਆਈ 1.1.x
- ਮੈਵੇਨ
- ਲੈਂਗਚੇਨ4ਜ
- ਏਜ਼ੂਰ ਏਆਈ ਫਾਉਂਡਰੀ, ਏਜ਼ੂਰ ਓਪਨਏਆਈ, ਅਤੇ ਓਪਨਏਆਈ SDKs

**ਆਰਕੀਟੈਕਚਰ:**
- ਕੈਪਟਰਾਂ ਦੇ ਅਨੁਸਾਰ ਸੁਤੰਤਰ ਸਪ੍ਰਿੰਗ ਬੂਟ ਐਪਲੀਕੇਸ਼ਨਾਂ ਦੀ ਕ੍ਰਮਬੱਧਤਾ
- ਵੱਖ-ਵੱਖ ਏਆਈ ਪੈਟਰਨ ਦਿਖਾਉਂਦੇ ਸੈਂਪਲ ਪ੍ਰੋਜੈਕਟ
- ਗਿਟਹੱਬ ਕੋਡਸਪੇਸ-ਤਿਆਰ ਪਹਿਲਾਂ ਤੋਂ ਸੰਰਚਿਤ ਡੈਵ ਕੰਟੇਨਰਸ ਨਾਲ

## ਸੈਟਅੱਪ ਕਮਾਂਡਾਂ

### ਜਰੂਰੀਯਾਤ
- ਜਾਵਾ 21 ਜਾਂ ਇਸ ਤੋਂ ਉੱਚਾ
- ਮੈਵੇਨ 3.x
- ਏਜ਼ੂਰ ਸਬਸਕ੍ਰਿਪਸ਼ਨ ਜਿਸ ਵਿੱਚ ਏਜ਼ੂਰ ਏਆਈ ਫਾਉਂਡਰੀ ਮਾਡਲ ਡਿਪਲੋਏਮੈਂਟ ਹੈ (`azd up` ਨਾਲ ਪ੍ਰੋਵਿਜ਼ਨ ਕਰੋ)
- ਏਜ਼ੂਰ CLI (`az`) ਅਤੇ ਏਜ਼ੂਰ ਡਿਵੈਲਪਰ CLI (`azd`), ਬਿਨਾਂ ਕੰਜੀ ਲਾਗਿਨ ਲਈ ਸਾਈਨ-ਇਨ

```bash
# ਵਾਤਾਵਰਣ ਸੈੱਟਅਪ ਲਈ ਲਗਭਗ 2 ਮਿੰਟ ਉਡੀਕੋ
```

```bash
# ਜਦੋਂ ਮੰਗ ਕੀਤੀ ਜਾਏ ਤਾਂ ਕੰਟੇਨਰ ਵਿੱਚ ਮੁੜ ਖੋਲ੍ਹੋ
```

### ਸੰਰਚਨਾ

**ਏਜ਼ੂਰ ਏਆਈ ਫਾਉਂਡਰੀ ਸੈਟਅੱਪ (ਬਿਨਾਂ ਕੁੰਜੀ, ਸਿਫਾਰਸ਼ੀ):**
```bash
# ਫਾਉਂਡਰੀ ਖਾਤਾ + ਮਾਡਲ ਡਿਪਲੋਯਮੈਂਟ ਨੂੰ ਕੋਡ ਵਜੋਂ ਪ੍ਰਦਾਨ ਕਰੋ
cd 02-SetupDevEnvironment
azd auth login
az login
azd up
# azd ਤੁਹਾਡੇ ਐਂਡਪੌਇੰਟ (ਕੋਈ ਕੁੰਜੀ ਨਹੀਂ) ਨਾਲ examples/basic-chat-azure/.env ਲਿਖਦਾ ਹੈ
```

**ਮੈਨੂਅਲ ਐਂਡਪੋਇੰਟ ਕਨਫਿਗ:**
```bash
# ਜੇ ਤੁਸੀਂ azd ਵਰਤਿਆ ਨਹੀਂ, ਤਾਂ endpoint ਆਪ ਹੀ ਸੈੱਟ ਕਰੋ (auth az login ਰਾਹੀਂ ਬਿਨਾਂ ਕੁੰਜੀ ਦੇ ਰਹਿੰਦੀ ਹੈ)
cd 02-SetupDevEnvironment/examples/basic-chat-azure
cp .env.example .env
# .env ਸੋਧੋ: AZURE_OPENAI_ENDPOINT=https://<resource>.openai.azure.com/
```

```bash
cd [project-directory]
```

**ਕਲਾਈਂਟ ਉਦਾਹਰਣ ਚਲਾਉਣਾ:**
```bash
# ਕਿਸੇ ਹੋਰ ਟਰਮੀਨਲ ਵਿੱਚ ਸਰਵਰ ਸ਼ੁਰੂ ਕਰਨ ਤੋਂ ਬਾਅਦ
cd 04-PracticalSamples/calculator

# ਡਾਇਰੈਕਟ MCP ਕਲਾਇੰਟ
mvn exec:java -Dexec.mainClass="com.microsoft.mcp.sample.client.SDKClient"

# ਏਆਈ ਸਚਾਲਿਤ ਕਲਾਇੰਟ (AZURE_OPENAI_ENDPOINT + az ਲਾਗਿਨ ਦੀ ਲੋੜ)
mvn exec:java -Dexec.mainClass="com.microsoft.mcp.sample.client.LangChain4jClient"

# ਇੰਟਰਐਕਟਿਵ ਬੌਟ
mvn exec:java -Dexec.mainClass="com.microsoft.mcp.sample.client.Bot"
```

## ਕੋਡ ਸ਼ੈਲੀ ਦੇ ਨਿਯਮ

### ਜਾਵਾ ਰਿਵਾਜ਼
- **ਜਾਵਾ ਵਰਜਨ:** ਜਾਵਾ 21 ਆਧੁਨਿਕ ਖੂਬੀਆਂ ਨਾਲ
- **ਸਟਾਈਲ:** ਮਿਆਰੀ ਜਾਵਾ ਰਿਵਾਜ਼ਾਂ ਦੀ ਪਾਲਨਾ ਕਰੋ
- **ਨਾਮ ਕਿਰਿਆ:** 
  - ਕਲਾਸ: PascalCase
  - ਮੇਥਡ/ਵੈਰੀਏਬਲ: camelCase
  - ਕਾਂਸਟੈਂਟ: UPPER_SNAKE_CASE
  - ਪੈਕੇਜ ਨਾਮ: ਛੋਟੇ ਅੱਖਰ

### ਸਪ੍ਰਿੰਗ ਬੂਟ ਪੈਟਰਨ
- ਕਾਰੋਬਾਰੀ ਲਾਜਿਕ ਲਈ `@Service` ਵਰਤੋ
- REST ਐਂਡਪੋਇੰਟ ਲਈ `@RestController` ਵਰਤੋ
- ਸੰਰਚਨਾ `application.yml` ਜਾਂ `application.properties` ਰਾਹੀਂ
- ਕਠੋਰ ਪਾਠੀਆਂ ਵੱਲੋਂ ਵਾਤਾਵਰਨ ਵੈਰੀਏਬਲਾਂ ਨੂੰ ਤਰਜੀਹ
- MCP ਲਈ ਖੁਲ੍ਹੇ ਮੈਥਡਾਂ `@Tool` ਐਨੋਟੇਸ਼ਨ ਨਾਲ

### ਵਾਤਾਵਰਣ ਅਨੁਸਾਰ ਸੰਰਚਨਾ

**ਡਿਵੈਲਪਮੈਂਟ:**
```yaml
# application.yml (keyless - no api-key; auth via DefaultAzureCredential)
spring:
  ai:
    azure:
      openai:
        endpoint: ${AZURE_OPENAI_ENDPOINT}
        chat:
          options:
            deployment-name: ${AZURE_OPENAI_DEPLOYMENT:gpt-4o-mini}
```

**ਪ੍ਰੋਡਕਸ਼ਨ:**
- ਬਿਨਾਂ ਕੁੰਜੀ ਲਈ `az login` ਦੀ ਥਾਂ ਇੱਕ ਪ੍ਰਬੰਧਿਤ ਪਹਚਾਣ ਵਰਤੋ
- ਆਪਣੇ ਪ੍ਰੋਡਕਸ਼ਨ ਫਾਉਂਡਰੀ ਸਰੋਤ ਵੱਲ `AZURE_OPENAI_ENDPOINT` ਨੂੰ ਪੌਇੰਟ ਕਰੋ
- ਸੰਰਚਨਾ ਵਾਤਾਵਰਣ ਵੈਰੀਏਬਲਾਂ ਜਾਂ ਏਜ਼ੂਰ ਕੀ ਵੋਲਟ ਰਾਹੀਂ ਸੰਭਾਲੋ

- ਵਿਸ਼ੇਸ਼ ਡਿਪਲੋਇਮੈਂਟ ਨੋਟਾਂ ਲਈ ਹਰੇਕ ਪ੍ਰੋਜੈਕਟ ਦੇ README ਵੇਖੋ

## ਵਧੂ ਨੋਟਸ

### ਏਜ਼ੂਰ ਏਆਈ ਫਾਉਂਡਰੀ
- **ਬਿਨਾਂ ਕੁੰਜੀ ਪ੍ਰਮਾਣਿਕਤਾ:** ਮਾਈਕ੍ਰੋਸੌਫਟ ਏਂਟਰਾ ID ਨਾਲ ਜੁੜੋ — ਕੋਈ API ਕੁੰਜੀ ਪ੍ਰਬੰਧਨ ਨਹੀਂ
- **ਕੋਡ ਦੇ ਤੌਰ 'ਤੇ ਪ੍ਰੋਵਿਜ਼ਨਿੰਗ:** ਬਾਈਸੈਪ + azd (`azd up`) ਖਾਤਾ ਅਤੇ ਮਾਡਲ ਡਿਪਲੋਏਮੈਂਟ ਬਣਾਉਂਦੇ ਹਨ
- ਇਹ ਉਹੀ ਓਪਨਏਆਈ-ਕੰਪੈਟਿਬਲ ਕੋਡ ਸਥਾਨਕ (`az login`) ਅਤੇ ਏਜ਼ੂਰ (ਪ੍ਰਬੰਧਿਤ ਪਹਚਾਣ) ਵਿੱਚ ਦੌੜਦਾ ਹੈ

### ਕਈ ਪ੍ਰੋਜੈਕਟਾਂ ਨਾਲ ਕੰਮ ਕਰਨਾ
ਹਰ ਸੈਂਪਲ ਪ੍ਰੋਜੈਕਟ ਸੁਤੰਤਰ ਹੈ:
```bash
# ਖਾਸ ਪ੍ਰੋਜੈਕਟ ਤੇ ਜਾਓ
cd 04-PracticalSamples/[project-name]

# ਹਰ ਇੱਕ ਦਾ ਆਪਣਾ pom.xml ਹੁੰਦਾ ਹੈ ਅਤੇ ਇਨ੍ਹਾਂ ਨੂੰ ਸਵਤੰਤਰ ਤੌਰ 'ਤੇ ਬਣਾਇਆ ਜਾ ਸਕਦਾ ਹੈ
mvn clean install
```

### ਆਮ ਸਮੱਸਿਆਵਾਂ

**ਜਾਵਾ ਵਰਜਨ ਮੇਲ-ਖਾਂਹ:**
```bash
# ਜਾਵਾ 21 ਦੀ ਜਾਂਚ ਕਰੋ
java -version
# ਜਦੋਂ ਜਰੂਰੀ ਹੋਵੇ ਤਾਂ JAVA_HOME ਨੂੰ ਅਪਡੇਟ ਕਰੋ
export JAVA_HOME=/usr/lib/jvm/msopenjdk-current
```

**ਨਿਰਭਰਤਾ ਡਾਊਨਲੋਡ ਸਮੱਸਿਆਵਾਂ:**
```bash
# ਮੈਵਨ ਕੈਸ਼ ਸਾਫ਼ ਕਰੋ ਅਤੇ ਦੁਬਾਰਾ ਕੋਸ਼ਿਸ਼ ਕਰੋ
rm -rf ~/.m2/repository
mvn clean install
```

**ਐਂਡਪੋਇੰਟ ਜਾਂ ਸਾਈਨ-ਇਨ ਨਹੀਂ ਮਿਲਿਆ:**
```bash
# ਵਰਤਮਾਨ ਸੈਸ਼ਨ ਵਿੱਚ ਐਂਡਪੋਇੰਟ ਸੈੱਟ ਕਰੋ ਅਤੇ ਸਾਈਨ ਇਨ ਕਰੋ (ਬਿਨਾਂ ਕੁੰਜੀ ਦੇ)
export AZURE_OPENAI_ENDPOINT="https://your-resource.openai.azure.com/"
az login

# ਜਾਂ ਪ੍ਰਾਜੈਕਟ ਡਾਇਰੈਕਟਰੀ ਵਿੱਚ .env ਫਾਇਲ ਦੀ ਵਰਤੋਂ ਕਰੋ
echo "AZURE_OPENAI_ENDPOINT=https://your-resource.openai.azure.com/" > .env
```

**ਪੋਰਟ ਪਹਿਲਾਂ ਹੀ ਵਰਤਮਾਨ ਹੈ:**
```bash
# ਸਪ੍ਰਿੰਗ ਬੂਟ ਮੂਲ ਰੂਪ ਵਿੱਚ ਪੋਰਟ 8080 ਵਰਤਦਾ ਹੈ
# application.properties ਵਿੱਚ ਬਦਲاؤ:
server.port=8081
```
