# التطبيقات العملية والمشاريع

## ما ستتعلمه
في هذا القسم سنعرض ثلاثة تطبيقات عملية توضح أنماط تطوير الذكاء الاصطناعي التوليدي باستخدام جافا:
- إنشاء مُولد قصص للحيوانات الأليفة متعدد الوسائط يجمع بين الذكاء الاصطناعي على جانب العميل والجانب الخادمي
- تنفيذ دمج نموذج ذكاء اصطناعي محلي مع عرض Foundry Local Spring Boot التوضيحي
- تطوير خدمة بروتوكول سياق النموذج (MCP) باستخدام مثال الآلة الحاسبة

## مقدمة

هذا الفصل يعرض **مشاريع نموذجية** توضح أنماط تطوير الذكاء الاصطناعي التوليدي باستخدام جافا. كل مشروع وظيفي بالكامل ويبين تقنيات ذكاء اصطناعي محددة وأنماط معمارية وأفضل الممارسات التي يمكنك تكييفها لتطبيقاتك الخاصة.

### عرض Foundry Local Spring Boot التوضيحي

يُظهر **[عرض Foundry Local Spring Boot التوضيحي](foundrylocal/README.md)** كيفية التكامل مع نماذج ذكاء اصطناعي محلية باستخدام **OpenAI Java SDK**. يعرض الاتصال بالنماذج التي تعمل على Foundry Local (مثل **Phi-4-mini**)، مع الكشف التلقائي عن النموذج، مما يسمح بتشغيل تطبيقات الذكاء الاصطناعي دون الاعتماد على خدمات السحابة.

### مُولد قصص الحيوانات الأليفة

يُعد **[مُولد قصص الحيوانات الأليفة](petstory/README.md)** تطبيق ويب رائع باستخدام Spring Boot يوضح **المعالجة متعددة الوسائط للذكاء الاصطناعي** لتوليد قصص مبدعة عن الحيوانات الأليفة. يجمع بين قدرات الذكاء الاصطناعي على جانب العميل والخادم باستخدام transformer.js للتفاعل مع الذكاء الاصطناعي في المتصفح و OpenAI SDK للمعالجة على الخادم.

### خدمة MCP للآلة الحاسبة (عرض MCP مناسب للمبتدئين)

يُعد **[خدمة MCP للآلة الحاسبة](calculator/README.md)** عرضًا مبسطًا لـ **بروتوكول سياق النموذج (MCP)** باستخدام Spring AI. يوفر مقدمة مناسبة للمبتدئين لمفاهيم MCP، ويُظهر كيف تنشئ خادم MCP أساسي يتفاعل مع عملاء MCP.

## تقدم التعلم

تم تصميم هذه المشاريع للبناء على مفاهيم الفصول السابقة:

1. **ابدأ ببساطة**: ابدأ بعرض Foundry Local Spring Boot التوضيحي لفهم التكامل الأساسي مع نماذج محلية  
2. **أضف التفاعلية**: تقدم إلى مُولد قصص الحيوانات الأليفة للذكاء الاصطناعي متعدد الوسائط والتفاعل عبر الويب  
3. **تعلم أساسيات MCP**: جرب خدمة MCP للآلة الحاسبة لفهم أساسيات بروتوكول سياق النموذج  

## ملخص

عمل جيد! لقد استعرضت الآن بعض التطبيقات الواقعية:

- تجارب ذكاء اصطناعي متعددة الوسائط تعمل في المتصفح وعلى الخادم  
- دمج نموذج ذكاء اصطناعي محلي باستخدام أطر عمل وجافا SDK حديثة  
- خدمتك الأولى لبروتوكول سياق النموذج لرؤية كيفية تكامل الأدوات مع الذكاء الاصطناعي
