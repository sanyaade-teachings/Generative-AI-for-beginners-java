# бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ“бҹҚбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„ бһ“бһ·бһ„бһӮбһҳбҹ’бһҡбҹ„бһ„

## бһўбҹ’бһңбһёбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҡбҹҖбһ“
бһҖбҹ’бһ“бһ»бһ„бһ•бҹ’бһ“бҹӮбһҖбһ“бҹҒбҹҮ бһҷбһҫбһ„бһ“бһ№бһ„бһ”бһ„бҹ’бһ бһ¶бһүбһҖбһ¶бһҡбһ”бһ„бҹ’бһ бһ¶бһүбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ“бҹҚбһ”бһёбһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһӮбҹҶбһҡбһјбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚ AI бһ–бһёбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҮбһ¶бһҳбһҪбһҷ Javaбҹ–
- бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ…бһҖбҹ’бһҡбһңбһ…бһ“бҹ’бһҠбһҡбһҹбһҸбҹ’бһңбһҠбҹӮбһӣбһҳбһ¶бһ“бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҳбһ»бһҒбһ„бһ¶бһҡбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“ бһҠбҹӮбһӣбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ AI бһҳбһ»бһҒбһ„бһ¶бһҡбһўбһҸбһ·бһҗбһ·бһҮбһ“ бһ“бһ·бһ„бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ
- бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҳбҹүбһјбһҠбҹӮбһӣ AI бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҮбһ¶бһҳбһҪбһҷбһ‘бҹҶбһ–бҹҗбһҡбһ”бһ„бҹ’бһ бһ¶бһү Foundry Local Spring Boot
- бһўбһ—бһ·бһңбһҢбҹ’бһҚбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ Model Context Protocol (MCP) бһҮбһ¶бһҳбһҪбһҷбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ Calculator

## бһ•бҹ’бһ‘бһ¶бҹҶбһ„бһ”бһ„бҹ’бһ бһ¶бһү

бһҮбҹҶбһ–бһјбһҖбһ“бҹҒбҹҮбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһё **бһӮбһҳбҹ’бһҡбҹ„бһ„бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ** бһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһӮбҹҶбһҡбһјбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚ AI бһ–бһёбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҮбһ¶бһҳбһҪбһҷ Javaбҹ” бһӮбһҳбҹ’бһҡбҹ„бһ„бһҳбһҪбһҷбҹ—бһҳбһ¶бһ“бһҳбһ»бһҒбһ„бһ¶бһҡбһ•бҹ’бһҸбһ¶бһ…бҹӢбһҳбһ»бһҒ бһ“бһ·бһ„бһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶ AI бһ•бҹ’бһҹбҹҒбһ„бҹ— бһӮбҹҶбһҡбһјбһҹбҹҶбһҺбһ„бҹӢ бһ“бһ·бһ„бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһӣбҹ’бһўбҹ—бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҠбһҫбһҳбҹ’бһ”бһёбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

### Demo Foundry Local Spring Boot

**[Demo Foundry Local Spring Boot](foundrylocal/README.md)** бһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһҡбһ”бҹҖбһ”бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҮбһ¶бһҳбһҪбһҷбһҳбҹүбһјбһҠбҹӮбһӣ AI бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ **OpenAI Java SDK**бҹ” бһңбһ¶бһ”бһ„бҹ’бһ бһ¶бһүбһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ…бһҳбҹүбһјбһҠбҹӮбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ“бҹ…бһӣбһҫ Foundry Local (бһҠбһјбһ…бһҮбһ¶ **Phi-4-mini**), бһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҳбҹүбһјбһҠбҹӮбһӣбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·, бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһҡбһҸбҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё AI бһҠбҹ„бһҷбһҳбһ·бһ“бһҸбҹ’бһҡбһјбһңбһ–бһ№бһ„бһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһҳбҹҒбһғбҹ”

### бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ…бһҖбҹ’бһҡбһңбһ…бһ“бҹ’бһҠбһҡбһҹбһҸбҹ’бһң

**[бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ…бһҖбҹ’бһҡбһңбһ…бһ“бҹ’бһҠбһҡбһҹбһҸбҹ’бһң](petstory/README.md)** бһҮбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһҺбҹ’бһҸбһ¶бһү Spring Boot бһҠбҹӮбһӣбһӮбһҪбһҡбһұбҹ’бһҷбһ…бһ¶бһ”бҹӢбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбһҳбһҪбһҷ бһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһё **бһҖбҹҶбһҺбһҸбҹӢбһ бҹҒбһҸбһ»бһ–бһ бһ»бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ AI** бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһҝбһ„бһўбҹҶбһ–бһёбһҹбһҸбҹ’бһңбһҠбҹӮбһӣбһҳбһ¶бһ“бһҖбһ¶бһҡбһ…бҹ’бһ“бҹғбһ”бҹ’бһҡбһҢбһ·бһҸбҹ” бһңбһ¶бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ– AI бһҳбһ»бһҒбһ„бһ¶бһҡбһўбһҸбһ·бһҗбһ·бһҮбһ“ бһ“бһ·бһ„бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ transformer.js бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбһ“бҹ’бһҸбһҡбһҖбһҳбҹ’бһҳ AI бһ“бҹ…бһӣбһҫбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖ бһ“бһ·бһ„ OpenAI SDK бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ–бһёбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫбҹ”

### бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһҖбһ¶бһӣбҹӢбһӮбһ»бһӣбҹҒбһҸ MCP (Demo MCP бһҹбһҳбҹ’бһҡбһҪбһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ)

**[бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһҖбһ¶бһӣбҹӢбһӮбһ»бһӣбҹҒбһҸ MCP](calculator/README.md)** бһҮбһ¶бһҖбһ¶бһҡбһ”бһ„бҹ’бһ бһ¶бһүбһҹбһ¶бһҳбһүбҹ’бһүбһ“бҹғ **Model Context Protocol (MCP)** бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ Spring AIбҹ” бһңбһ¶бһ”бһ„бҹ’бһ бһ¶бһүбһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһҗбҹ’бһ“бһ¶бһҖбҹӢбһ”бһ„бҹ’бһ бһ¶бһүбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ MCP бһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһҡбһ”бҹҖбһ”бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ MCP бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҠбҹӮбһӣбһўбһ“бҹ’бһҸбһҡбһҖбһҳбҹ’бһҳбһҮбһ¶бһҳбһҪбһҷбһўбһҸбһ·бһҗбһ·бһҮбһ“ MCPбҹ”

## бһҖбһ¶бһҡбһҡбһёбһҖбһ…бһҳбҹ’бһҡбһҫбһ“бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҡбҹҖбһ“

бһӮбһҳбҹ’бһҡбҹ„бһ„бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбһ…бһ“бһ¶бһЎбһҫбһ„бһҠбһҫбһҳбҹ’бһ”бһёбһ…бһ¶бһ”бҹӢбһ–бһёбһӮбҹҶбһ“бһ·бһҸбһҖбҹ’бһ“бһ»бһ„бһҮбҹҶбһ–бһјбһҖбһҳбһ»бһ“бҹ—бҹ–

1. **бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҠбҹ„бһҷбһҹбһ¶бһҳбһүбҹ’бһү**: бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҮбһ¶бһҳбһҪбһҷ Foundry Local Spring Boot Demo бһҠбһҫбһҳбҹ’бһ”бһёбһҷбһӣбҹӢбһғбһҫбһүбһ–бһёбһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ AI бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҮбһ¶бһҳбһҪбһҷбһҳбҹүбһјбһҠбҹӮбһӣбһҖбҹ’бһ“бһ»бһ„бһҹбҹ’бһҡбһ»бһҖ
2. **бһ”бһ“бҹ’бһҗбҹӮбһҳбһўбһ“бҹ’бһҸбһҡбһҖбһҳбҹ’бһҳ**: бһҡбһёбһҖбһ…бһҳбҹ’бһҡбһҫбһ“бһ‘бҹ…бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ…бһҖбҹ’бһҡбһңбһ…бһ“бҹ’бһҠбһҡбһҹбһҸбҹ’бһңбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ AI бһ–бһ бһ»бһҳбһ»бһҒбһ„бһ¶бһҡ бһ“бһ·бһ„бһўбһ“бҹ’бһҸбһҡбһҖбһҳбҹ’бһҳбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһ”бһҺбҹ’бһҸбһ¶бһү
3. **бһҡбҹҖбһ“бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ MCP**: бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһҖбһ¶бһӣбҹӢбһӮбһ»бһӣбҹҒбһҸ MCP бһҠбһҫбһҳбҹ’бһ”бһёбһҷбһӣбҹӢбһ–бһёбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ“бҹғ Model Context Protocol

## бһҹбһ„бҹ’бһҒбҹҒбһ”

бһӣбҹ’бһўбһҺбһ¶бһҹбҹӢ! бһҘбһЎбһјбһңбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһўбҹҶбһ–бһёбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„бһҒбҹ’бһӣбҹҮбҹ—бҹ–

- бһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚ AI бһ–бһ бһ»бһҳбһ»бһҒбһ„бһ¶бһҡбһҠбҹӮбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ‘бһ¶бҹҶбһ„бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖ бһ“бһ·бһ„бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ
- бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҳбҹүбһјбһҠбҹӮбһӣ AI бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҹбҹ’бһ“бһјбһӣ Java бһ“бһ·бһ„ SDK бһҠбҹӮбһӣбһ‘бҹҶбһ“бһҫбһ”
- бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ Model Context Protocol бһҠбҹҶбһ”бһјбһ„бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһҠбһҫбһҳбҹ’бһ”бһёбһҳбһҫбһӣбһҗбһ¶бһ§бһ”бһҖбһҡбһҺбҹҚбһўбһ¶бһ…бһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҮбһ¶бһҳбһҪбһҷ AI бһҠбһјбһ…бһҳбҹ’бһҸбҹҒбһ…
