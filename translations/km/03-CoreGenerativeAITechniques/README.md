## бһҸбһ¶бһҡбһ¶бһ„бһҳбһ¶бһҸбһ·бһҖбһ¶

- [бһӣбһҖбҹ’бһҒбһҺбҹҲбһҷбҹүбһ¶бһ„бһҸбҹ’бһҡбһјбһңбһҳбһ¶бһ“](#бһӣбһҖбҹ’бһҒбһҺбҹҲбһҷбҹүбһ¶бһ„бһҸбҹ’бһҡбһјбһңбһҳбһ¶бһ“)
- [бһҖбһ¶бһҡбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ](#бһҖбһ¶бһҡбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ)
  - [бһҮбҹҶбһ бһ¶бһ“ 1: бһҖбҹҶбһҺбһҸбҹӢбһ…бҹҶбһҺбһ»бһ…бһ”бһүбҹ’бһ…бһ”бҹӢ Foundry бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ](#бһҮбҹҶбһ бһ¶бһ“-1-бһҖбҹҶбһҺбһҸбҹӢбһ…бҹҶбһҺбһ»бһ…бһ”бһүбҹ’бһ…бһ”бҹӢ-foundry-бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ)
  - [бһҮбҹҶбһ бһ¶бһ“ 2: бһҶбҹ’бһӣбһ„бһҖбһ¶бһҸбҹӢбһҗбһҸбһҜбһҖбһҹбһ¶бһҡбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ](#бһҮбҹҶбһ бһ¶бһ“-2-бһҶбҹ’бһӣбһ„бһҖбһ¶бһҸбҹӢбһҗбһҸбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ)
- [бһҳбһӮбҹ’бһӮбһ»бһ‘бҹҒбһҹбһҖбҹҚбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҳбҹүбһјбһҠбҹӮбһӣ](#бһҳбһӮбҹ’бһӮбһ»бһ‘бҹҒбһҹбһҖбҹҚбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҳбҹүбһјбһҠбҹӮбһӣ)
- [бһҳбҹҒбһҡбҹҖбһ“ 1бҹ– бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһ”бҹӢ LLM бһ“бһ·бһ„бһҖбһ¶бһҡбһҹбһ“бҹ’бһ‘бһ“бһ¶](#бһҳбҹҒбһҡбҹҖбһ“-1бҹ–-бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһ”бҹӢ-llm-бһ“бһ·бһ„бһҖбһ¶бһҡбһҹбһ“бҹ’бһ‘бһ“бһ¶)
- [бһҳбҹҒбһҡбҹҖбһ“ 2бҹ– бһҖбһ¶бһҡбһ бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡ](#бһҳбҹҒбһҡбҹҖбһ“-2бҹ–-бһҖбһ¶бһҡбһ бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡ)
- [бһҳбҹҒбһҡбҹҖбһ“ 3бҹ– RAG (бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһ”бһ“бҹ’бһҗбҹӮбһҳбһҠбҹ„бһҷбһҷбһҖбһҳбһҖбһ–бһёбһҖбһ¶бһҡбһҹбҹ’бһңбҹӮбһ„бһҡбһҖ)](#бһҳбҹҒбһҡбҹҖбһ“-3бҹ–-rag-бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһ”бһ“бҹ’бһҗбҹӮбһҳбһҠбҹ„бһҷбһҷбһҖбһҳбһҖбһ–бһёбһҖбһ¶бһҡбһҹбҹ’бһңбҹӮбһ„бһҡбһҖ)
- [бһҳбҹҒбһҡбҹҖбһ“ 4бҹ– AI бһҳбһ¶бһ“бһҖбһ¶бһҸбһ–бҹ’бһңбһҖбһ·бһ…бҹ’бһ…](#бһҳбҹҒбһҡбҹҖбһ“-4бҹ–-ai-бһҳбһ¶бһ“бһҖбһ¶бһҸбһ–бҹ’бһңбһҖбһ·бһ…бҹ’бһ…)
- [бһӣбҹҶбһҠбһ¶бһ”бҹӢбһ‘бһјбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ](#бһӣбҹҶбһҠбһ¶бһ”бҹӢбһ‘бһјбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ)
- [бһҮбҹҶбһ бһ¶бһ“бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢ](#бһҮбҹҶбһ бһ¶бһ“бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢ)

## бһ‘бһ·бһҠбҹ’бһӢбһ—бһ¶бһ–бһ‘бһјбһӣбҹҶбһ‘бһјбһӣбһ¶бһҷ

бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбһ•бҹ’бһҸбһӣбҹӢбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҠбҹ„бһҷбһҠбҹғбһӣбһҫбһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹ AI бһ”бһ„бҹ’бһҖбһҫбһҸбһӮбһјбһҡбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ Java бһ“бһ·бһ„ Azure AI Foundryбҹ” бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҡбҹҖбһ“бһ–бһёбһҡбһ”бҹҖбһ”бһўбһ“бһ»бһңбһҸбҹ’бһҸбһўбһ“бҹ’бһҸбһҡбһҖбһҳбҹ’бһҳбһҮбһ¶бһҳбһҪбһҷбһҳбҹүбһјбһҠбҹӮбһӣбһ—бһ¶бһҹбһ¶вҖӢбһ’бҹҶ (LLMs), бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһ¶бһҡбһ бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡ, бһ”бҹ’бһҡбһҫбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһ”бһ“бҹ’бһҗбҹӮбһҳбһҠбҹ„бһҷбһҷбһҖбһҳбһҖбһ–бһёбһҖбһ¶бһҡбһҹбҹ’бһңбҹӮбһ„бһҡбһҖ (RAG), бһ“бһ·бһ„бһўбһ“бһ»бһңбһҸбҹ’бһҸбһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚ AI бһҳбһ¶бһ“бһҖбһ¶бһҸбһ–бҹ’бһңбһҖбһ·бһ…бҹ’бһ…бҹ”

## бһӣбһҖбҹ’бһҒбһҺбҹҲбһҷбҹүбһ¶бһ„бһҸбҹ’бһҡбһјбһңбһҳбһ¶бһ“

бһҳбһ»бһ“бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ бһҹбһјбһҳбһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶бһўбҹ’бһ“бһҖбһҳбһ¶бһ“бҹ–
- Java 21 бһ¬бһҒбҹ’бһ–бһҹбҹӢбһҮбһ¶бһ„бһ“бҹҒбҹҮбһ”бһ¶бһ“бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ
- Maven бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһӮбҹ’бһҡбһҝбһ„бһ…бһ„бһ—бҹ’бһҮбһ¶бһ”бҹӢ
- бһҖбһ¶бһҡбһ…бҹӮбһҖбһ…бһ¶бһҷбһҳбҹүбһјбһҠбҹӮбһӣ Azure AI Foundry бһҳбһҪбһҷ (бһ…бҹҒбһүбһңбһ¶ бһҮбһ¶бһҳбһҪбһҷ `azd up` вҖ” бһҳбһҫбһӣ [бһҮбҹҶбһ–бһјбһҖ 2](../02-SetupDevEnvironment/getting-started-azure-openai.md))
- [Azure CLI](https://learn.microsoft.com/cli/azure/install-azure-cli) бһ”бһ¶бһ“бһ…бһ»бҹҮбһҲбҹ’бһҳбҹ„бҹҮбһҮбһ¶бһҳбһҪбһҷ `az login` (Authentication бһҠбҹ„бһҷбһӮбҹ’бһҳбһ¶бһ“ key)

## бһҖбһ¶бһҡбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ

> **бһңбһ·бһ’бһёбһӣбһҝбһ“бһ”бҹҶбһ•бһ»бһҸ вҖ” бһҡбһҸбҹӢбһҖбҹ’бһ“бһ»бһ„ VS Code (F5):** бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһё `azd up` (бһҮбҹҶбһ–бһјбһҖ 2) бһ“бһ·бһ„ `az login` бһ”бһҫбһҖ **Run and Debug** (`Ctrl+Shift+D`), бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ config бһҳбһҪбһҷбһҠбһјбһ…бһҮбһ¶ **Ch03: LLM Completions & Chat**, бһ бһҫбһҷбһ…бһ»бһ… **F5**бҹ” бһ…бҹҶбһҺбһ»бһ…бһ”бһүбҹ’бһ…бһ”бҹӢбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһ•бҹ’бһ‘бһ»бһҖбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһ–бһё `.env` бһҠбҹӮбһӣ `azd up` бһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸ вҖ” бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҡбҹҶбһӣбһ„бһҮбҹҶбһ бһ¶бһ“ 1 бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ” бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҹбһ“бҹ’бһ‘бһ“бһ¶бһ•бҹ’бһ‘бһ¶бһӣбҹӢ бһўбҹ’бһ“бһҖбһңбһ¶бһҷбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ terminal бһ бһҫбһҷбһ”бһүбҹ’бһ…бһјбһӣ `exit` бһҠбһҫбһҳбҹ’бһ”бһёбһ…бһ¶бһҖбһ…бҹҒбһүбҹ” Configs бһҡбһҸбҹӢбһҖбҹ’бһ“бһ»бһ„бһ–бҹҒбһӣбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„бһҳбһ¶бһ“бһ“бҹ… [`.vscode/launch.json`](../../../.vscode/launch.json)бҹ”
>
> бһ…бһјбһӣбһ…бһ·бһҸбҹ’бһҸбһ”бһүбҹ’бһҮбһ¶бһӣбҹҒбһҒ? бһҸбһ¶бһҳбһҮбҹҶбһ бһ¶бһ“ 1 бһ“бһ·бһ„ бһҮбҹҶбһ бһ¶бһ“ 2 бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ”

### бһҮбҹҶбһ бһ¶бһ“ 1: бһҖбҹҶбһҺбһҸбҹӢбһ…бҹҶбһҺбһ»бһ…бһ”бһүбҹ’бһ…бһ”бҹӢ Foundry бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ

бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһ‘бҹ… Azure AI Foundry бһҮбһ¶бһҳбһҪбһҷ **Authentication бһӮбҹ’бһҳбһ¶бһ“ key** (Microsoft Entra ID)бҹ” бһ…бһјбһӣбһҠбҹ„бһҷ `az login` бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһҖбҹҶбһҺбһҸбҹӢбһ…бҹҶбһҺбһ»бһ…бһ”бһүбҹ’бһ…бһ”бҹӢ Foundry бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҮбһ¶бһ•бҹ’бһ‘бҹғбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһ…бҹӮбһҖбһ…бһ¶бһҷбһҮбһ¶бһҳбһҪбһҷ `azd up`, бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһҸбһҳбҹ’бһӣбҹғбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ `azd env get-value AZURE_OPENAI_ENDPOINT`бҹ”

**Windows (Command Prompt):**
```cmd
set AZURE_OPENAI_ENDPOINT=https://your-resource.openai.azure.com/
```

**Windows (PowerShell):**
```powershell
$env:AZURE_OPENAI_ENDPOINT="https://your-resource.openai.azure.com/"
```

**Linux/macOS:**
```bash
export AZURE_OPENAI_ENDPOINT=https://your-resource.openai.azure.com/
```

> бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ”бҹ’бһҡбһҫ `gpt-4o-mini` бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ…бҹӮбһҖбһ…бһ¶бһҷбһӣбҹҶбһ“бһ¶бҹҶбһҠбһҫбһҳбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһӣбһ»бһ”бһ”бҹҶбһ”бҹ’бһҠбһјбһҡбһ”бһ¶бһ“бһҮбһ¶бһҳбһҪбһҷбһўбһҗбҹҒбһҡвҖӢбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹ `AZURE_OPENAI_DEPLOYMENT`бҹ”

бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ“бҹҒбҹҮбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһӮбһ“бҹ’бһӣбһ„бһҹбҹ’бһ“бһјбһӣбһ“бҹғбһўбһ“бҹ’бһҸбһҡбһҖбһҳбҹ’бһҳбһҳбҹүбһјбһҠбҹӮбһӣбһ—бһ¶бһҹбһ¶бһ’бҹҶ (LLM) бһҸбһ¶бһҳбһҡбһҷбҹҲ Azure OpenAI API, бһҡбһҪбһҳбһҳбһ¶бһ“ initialization бһӮбҹ’бһҳбһ¶бһ“ key бһҮбһ¶бһҳбһҪбһҷ Azure AI Foundry, бһҠбҹҶбһҺбһҫбһҡбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҹбһ¶бһҡвҖӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһӣбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ, бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҹбһ“бҹ’бһ‘бһ“бһ¶вҖӢбһҸбһ¶бһҳбһҖбһ¶бһҡбһ”бҹ’бһҡбһҳбһјбһӣбһ•бҹ’бһҸбһ»бҹҶбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҹбһ¶бһҡ, бһ“бһ·бһ„бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ”бҹ’бһҡбһңбҹӮбһ„бһ…бһҳбҹ’бһӣбһҫбһҷбһ“бһ·бһ„бһҖбһҳбҹ’бһҡбһ·бһҸбһҖбһ¶бһҡбһ…бҹ’бһ“бҹғбһ”бҹ’бһҡбһҢбһ·бһҸбҹ”

2. **бһҳбһ»бһҒбһ„бһ¶бһҡбһӮбһҺбһ“бһ¶**: AI бһҹбҹ’бһ“бһҫбһҖбһ¶бһҡбһӮбһҺбһ“бһ¶ (15% бһ“бҹғ 240), бһўбҹ’бһ“бһҖбһӮбһҺбһ“бһ¶, AI бһ”бһ„бҹ’бһ бһ¶бһүбһӣбһ‘бҹ’бһ’бһ•бһӣ

### бһўбҹ’бһңбһёбһҠбҹӮбһӣбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ“бҹҒбҹҮбһ”бһ„бҹ’бһҡбҹҖбһ“

RAG бһҮбһ¶бһҡбһ”бҹҖбһ”бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҖбһ¶бһҡбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһ—бһ¶бһҹбһ¶ бһҠбҹ„бһҷбһ”бһүбҹ’бһ…бһјбһӣбһ”бһҡбһ·бһ”бһ‘бһҜбһҖбһҹбһ¶бһҡбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ…бһ…бһјбһӣбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҹбҹ’бһ“бһҫбһҹбһ»бҹҶ AI, бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһҳбҹүбһјбһҠбҹӮбһӣбһ•бҹ’бһҸбһӣбҹӢбһ…бһҳбҹ’бһӣбһҫбһҷбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бҹ…бһӣбһҫбһ”бҹ’бһҡбһ—бһ–бһ…бҹҶбһҺбҹҒбҹҮбһҠбһ№бһ„бһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбһҷбҹүбһ¶бһ„бһҳбҹүбһҸбҹӢбһ…бһҸбҹӢ бһҮбҹҶбһ“бһҪбһҹбһҳбһ·бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһ¬бһ…бһ¶бһҹбҹӢбһҹбһҳбҹҗбһҷбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҒбҹ’бһӣбһҪбһ“бһңбһ¶, бһҒбһҺбҹҲбһҖбһ¶бһҡбһҖбһ¶бһҡбһ–бһ¶бһҡбһ–бҹ’бһҡбҹҶбһҠбҹӮбһ“бһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢбһҡбһңбһ¶бһ„бһҹбҹҶбһҺбһҪбһҡбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ“бһ·бһ„бһ”бҹ’бһҡбһ—бһ–бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһўбҹҶбһҺбһ¶бһ…бһҸбһ¶бһҳбһңбһ·бһ’бһ¶бһ“бһҖбһ¶бһҡвҖӢбһӣбһҖбҹ’бһҒбһҺбҹҲбһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹ promptбҹ”

#### 2. бһҖбһ¶бһҡбһҮбһ¶бһ”бҹӢбһ…бһ·бһҸбҹ’бһҸбһ”бһҠбһ·бһҹбҹҒбһ’
```java
private boolean isRefusalResponse(String response) {
    String lowerResponse = response.toLowerCase();
    String[] refusalPatterns = {
        "i can't assist with", "i cannot assist with",
        "sorry, i can't", "sorry, i cannot",
        "i'm unable to", "against my guidelines"
    };
    
    for (String pattern : refusalPatterns) {
        if (lowerResponse.contains(pattern)) {
            return true;
        }
    }
    return false;
}
```

### бһўбҹ’бһңбһёбһҠбҹӮбһӣбһҖбһҫбһҸбһҳбһ¶бһ“бһ–бҹҒбһӣбһўбҹ’бһ“бһҖбһҡбһҸбҹӢбһңбһ¶

2. **бһҖбһ¶бһҡбһ”бһҠбһ·бһҹбҹҒбһ’бһҷбҹүбһ¶бһ„бһ—бҹ’бһӣбҹҒбһ…бһ—бҹ’бһӣбһ¶бҹҶбһ„**: бһҳбҹүбһјбһҠбҹӮбһӣбһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһҮбһ¶бһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹбһ”бһҠбһ·бһҹбҹҒбһ’бһ…бҹҶбһ–бҹ„бҹҮбһҖбһ¶бһҡбһҹбҹ’бһ“бһҫбһҹбһ»бҹҶ ("бһҒбҹ’бһүбһ»бҹҶбһҳбһ·бһ“бһўбһ¶бһ…бһҮбһҪбһҷбһ”бһ¶бһ“") (бһ‘бһјбһ‘бҹ…бһҮбһ¶бһҳбһҪбһҷбһҳбҹүбһјбһҠбҹӮбһӣбһҹбһҳбҹҗбһҷбһ‘бҹҶбһ“бһҫбһ”)
