# व्यावहारिक अनुप्रयोगहरू र परियोजनाहरू

## तपाईंले के सिक्नुहुनेछ
यस अनुभागमा हामी तीन व्यावहारिक अनुप्रयोगहरू प्रदर्शन गर्नेछौँ जसले Java सँग सृजनात्मक AI विकास ढाँचाहरू देखाउँछन्:
- क्लाइन्ट-साइड र सर्भर-साइड AI संयोजन गरी मल्टि-मोडल पेट स्टोरी जेनेरेटर सिर्जना गर्नुहोस्
- Foundry Local Spring Boot डेमोसँग स्थानीय AI मोडेल एकीकरण लागू गर्नुहोस्
- क्याल्कुलेटर उदाहरणसँग मोडेल कन्टेक्स्ट प्रोटोकल (MCP) सेवा विकास गर्नुहोस्

## परिचय

यस अध्यायले Java सँग सृजनात्मक AI विकास ढाँचाहरू प्रदर्शन गर्ने **नमूना परियोजनाहरू** देखाउँछ। प्रत्येक परियोजना पूर्ण रूपमा संचालनयोग्य छ र तपाइँका आफ्नै अनुप्रयोगहरूमा अनुकूलन गर्न सकिने विशिष्ट AI प्रविधिहरू, वास्तुकला ढाँचाहरू, र उत्तम अभ्यासहरू देखाउँछ।

### Foundry स्थानीय Spring Boot डेमो

**[Foundry Local Spring Boot Demo](foundrylocal/README.md)** ले **OpenAI Java SDK** प्रयोग गरी स्थानीय AI मोडेलहरूसँग कसरी एकीकृत गर्ने देखाउँछ। यसले Foundry Local मा चलिरहेका मोडेलहरू (जस्तै, **Phi-4-mini**) मा जडान गर्ने तरिका देखाउँछ, स्वचालित मोडेल पहिचान सहित, जसले तपाईंलाई क्लाउड सेवाहरूमाथि निर्भर नहुने गरी AI अनुप्रयोगहरू चलाउन अनुमति दिन्छ।

### पेट स्टोरी जेनेरेटर

**[Pet Story Generator](petstory/README.md)** एक आकर्षक Spring Boot वेब अनुप्रयोग हो जसले **मल्टि-मोडल AI प्रशोधन** देखाउँदै सिर्जनात्मक पेट कथाहरू उत्पादन गर्दछ। यसले क्लाइन्ट-साइड र सर्भर-साइड AI क्षमता जम्मा गर्दछ, जसमा ब्राउजर-आधारित AI अन्तरक्रियाको लागि transformer.js र सर्भर-साइड प्रशोधनको लागि OpenAI SDK प्रयोग गरिएको छ।

### MCP क्याल्कुलेटर सेवा (शुरुआती मैत्री MCP डेमो)

**[MCP Calculator Service](calculator/README.md)** Spring AI प्रयोग गरेर **मोडेल कन्टेक्स्ट प्रोटोकल (MCP)** को सरल प्रदर्शन हो। यसले MCP अवधारणाहरूको लागि शुरुआती मैत्री परिचय दिन्छ, कसरी एउटा आधारभूत MCP सर्वर सिर्जना गर्ने र MCP क्लाइन्टहरूसँग अन्तरक्रिया गर्ने देखाउँछ।

## अध्ययन प्रगति

यी परियोजनाहरूलाई अघिल्लो अध्यायका अवधारणाहरूमा आधारित बनाइएको छ:

1. **सरलदेखि सुरु गर्नुहोस्**: Foundry Local Spring Boot Demo बाट शुरू गरी स्थानीय मोडेलहरूसँग आधारभूत AI एकीकरण बुझ्नुहोस्
2. **अन्तरक्रियात्मकता थप्नुहोस्**: Pet Story Generator तर्फ प्रवेश गरी मल्टि-मोडल AI र वेब-आधारित अन्तरक्रियाहरू सिक्नुहोस्
3. **MCP आधारभूतहरू सिक्नुहोस्**: MCP Calculator Service प्रयोग गरी मोडेल कन्टेक्स्ट प्रोटोकलका आधारभूत कुरा बुझ्नुहोस्

## सारांश

शानदार काम! तपाइँले अहिले केही वास्तविक अनुप्रयोगहरू अन्वेषण गर्नुभएको छ:

- ब्राउजर र सर्भर दुवैमा कार्य गर्ने मल्टि-मोडल AI अनुभवहरू
- आधुनिक Java फ्रेमवर्कहरू र SDKs प्रयोग गरी स्थानीय AI मोडेल एकीकरण
- तपाइँको पहिलो मोडेल कन्टेक्स्ट प्रोटोकल सेवा जसले AI सँग उपकरणहरू कसरी एकीकृत हुन्छन् देखाउँछ
