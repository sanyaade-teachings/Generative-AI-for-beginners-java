## Introducere

Acest capitol prezintă **proiecte exemplu** care demonstrează modele de dezvoltare AI generativă cu Java. Fiecare proiect este complet funcțional și evidențiază tehnologii AI, tipare arhitecturale și bune practici specifice pe care le puteți adapta pentru propriile aplicații.

### Demo Foundry Local Spring Boot

**[Foundry Local Spring Boot Demo](foundrylocal/README.md)** demonstrează cum să integrați modele AI locale folosind **OpenAI Java SDK**. Prezintă conectarea la modele ce rulează pe Foundry Local (de exemplu, **Phi-4-mini**), cu detectare automată a modelului, permițând rularea aplicațiilor AI fără dependență de servicii cloud.

### Generator de Povești cu Animale de Companie

**[Pet Story Generator](petstory/README.md)** este o aplicație web Spring Boot captivantă care demonstrează **procesare AI multi-modală** pentru generarea creativă de povești cu animale de companie. Combină capabilitățile AI pe client și pe server folosind transformer.js pentru interacțiuni AI în browser și SDK-ul OpenAI pentru procesare pe server.

### Serviciul MCP Calculator (Demo MCP Prietenos Pentru Începători)

**[Serviciul MCP Calculator](calculator/README.md)** este o demonstrație simplă a **Model Context Protocol (MCP)** folosind Spring AI. Oferă o introducere prietenoasă pentru începători în conceptele MCP, arătând cum să creați un server MCP de bază care interacționează cu clienți MCP.

## Progresul Învățării

Aceste proiecte sunt concepute să se bazeze pe conceptele din capitolele anterioare:

1. **Începe Simplu**: Începe cu Foundry Local Spring Boot Demo pentru a înțelege integrarea AI de bază cu modele locale
2. **Adaugă Interactivitate**: Progresează la Pet Story Generator pentru AI multi-modal și interacțiuni web
3. **Învață Bazele MCP**: Încearcă Serviciul MCP Calculator pentru a înțelege fundamentele Model Context Protocol

## Rezumat

Bravo! Acum ați explorat câteva aplicații reale:

- Experiențe AI multi-modale ce funcționează atât în browser, cât și pe server
- Integrare locală a modelelor AI folosind framework-uri Java moderne și SDK-uri
- Primul dvs. serviciu Model Context Protocol pentru a vedea cum se integrează uneltele cu AI
