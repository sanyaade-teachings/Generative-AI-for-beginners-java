# ప్రాక్టికల్ అప్లికేషన్లు & ప్రాజెక్టులు

## మీరు నేర్చుకునేది
ఈ విభాగంలో మేము జావాతో ఉత్పాదక AI అభివృద్ధి నమూనాలను ప్రదర్శించే మూడు ప్రాక్టికల్ అప్లికేషన్లను డెమో చేస్తాము:
- క్లయింట్-సైడ్ మరియు సర్వర్-సైడ్ AI కలిపి బహుముఖ్యమైన పెట్ స్టోరీ జనరేటర్ సృష్టించండి
- Foundry Local Spring Boot డెమోతో స్థానిక AI మోడల్ ఒకీకరణను అమలు చేయండి
- కాలిక్యులేటర్ ఉదాహరణతో మోడల్ కాంటెక్స్ట్ ప్రోటోకాల్ (MCP) సేవను అభివృద్ధి చేయండి

## పరిచయం

ఈ అధ్యాయం **జావాతో ఉత్పాదక AI అభివృద్ధి నమూనాలను** ప్రదర్శించే **నమూనా ప్రాజెక్టులను** చూపిస్తుంది. ప్రతి ప్రాజెక్టు పూర్తిగా పనితీరు కలిగి ఉంటుంది మరియు నిర్దిష్ట AI సాంకేతికతలు, వాస್ತುశిల్ప నమూనాలు, మరియు ఉత్తమ పద్ధతులను ప్రదర్శిస్తుంది, మీరు వాటిని మీ స్వంత అప్లికేషన్లకు అనుకూలపరచుకోవచ్చు.

### Foundry Local Spring Boot డెమో

**[Foundry Local Spring Boot డెమో](foundrylocal/README.md)** **OpenAI Java SDK** ఉపయోగించి స్థానిక AI మోడల్స్ తో ఎలా ఒకీకరించాలో ప్రదర్శిస్తుంది. ఇది Foundry Localపై నడుస్తున్న మోడల్స్ (ఉదా: **Phi-4-mini**)తో కనెక్ట్ అవ్వడాన్ని చూపిస్తుంది, ఆటోమేటిక్ మోడల్ గుర్తింపు తో, క్లౌడ్ సేవలపై ఆధారపడకుండా AI అప్లికేషన్లు నడపడానికి అనుమతిస్తుంది.

### పెట్ స్టోరీ జనరేటర్

**[పెట్ స్టోరీ జనరేటర్](petstory/README.md)** ఒక ఆకర్షణీయమైన Spring Boot వెబ్ అప్లికేషన్, ఇది సృజనాత్మక పెట్ కథలను ఉత్పత్తి చేయడానికి **బహుముఖ్య AI ప్రాసెసింగ్**ను చూపిస్తుంది. ఇది క్లయింట్-సైడ్ మరియు సర్వర్-సైడ్ AI సామర్థ్యాలను transformer.js ద్వారా బ్రౌజర్ ఆధారిత AI సంభాషణల కోసం మరియు OpenAI SDK ద్వారా సర్వర్-సైడ్ ప్రాసెసింగ్ కోసం కలుపుతుంది.

### MCP కాలిక్యులేటర్ సేవ (ప్రాథమిక MCP డెమో)

**[MCP కాలిక్యులేటర్ సేవ](calculator/README.md)** Spring AI ఉపయోగించి **Model Context Protocol (MCP)** యొక్క సాదారణ డెమో. ఇది MCP భావనలకు ప్రాథమిక పరిచయాన్ని అందిస్తుంది, MCP క్లయింట్లతో ఇంటరాక్ట్ అయ్యే ప్రాథమిక MCP సర్వర్ సృష్టించడం ఎలా చేయాలో చూపిస్తుంది.

## అధ్యయన ప్రగతి

ఈ ప్రాజెక్టులు మునుపటి అధ్యాయాల నుంచి భావనలను పెంచేలా రూపొందించబడ్డాయి:

1. **సాదరమైనదిగా ప్రారంభించండి**: స్థానిక మోడల్స్ తో ప్రాథమిక AI ఒకీకరణను అర్థం చేసుకోవడానికి Foundry Local Spring Boot డెమోతో ప్రారంభించండి
2. **సంబంధాన్ని జోడించండి**: బహుముఖ్య AI మరియు వెబ్ ఆధారిత సహకారాల కోసం పెట్ స్టోరీ జనరేటర్ వద్దకు వేగవంతం చేసుకోండి
3. **MCP ప్రాథమికాలను నేర్చుకోండి**: Model Context Protocol మౌలికాలను అర్థం చేసుకోవడానికి MCP కాలిక్యులేటర్ సేవను ప్రయత్నించండి

## సారాంశం

అద్భుతం! మీరు ఇప్పుడు కొన్ని నిజమైన అనువర్తనాలను తెలుసుకున్నారు:

- బ్రౌజర్ మరియు సర్వర్ రెండింటిలో పనిచేసే బహుముఖ్య AI అనుభవాలు
- ఆధునిక జావా ఫ్రేమ్‌వర్క్స్ మరియు SDKs ఉపయోగించి స్థానిక AI మోడల్ ఒకీకరణ
- మీరు AIతో టూల్స్ ఎలా ఒకీకరిస్తాయో చూడటానికి మీ మొదటి Model Context Protocol సేవ
